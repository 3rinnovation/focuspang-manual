# 2.6 과제관리

- **\[과제 관리]** 메뉴에서 등록한 과제를 관리할 수 있습니다.
- 과제 관리에서 **\[+과제 추가]** 클릭한 후 제목, 과제 공개 일시, 내용을 작성하고 등록합니다.
- **\[과제 관리]** 리스트의 **\[더보기]** 를 통해 과제 내용을 수정할 수 있고, **[삭제]** 버튼으로 삭제할 수도 있습니다.
