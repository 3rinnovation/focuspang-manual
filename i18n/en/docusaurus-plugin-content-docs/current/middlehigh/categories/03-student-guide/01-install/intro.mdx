# 1. Installation

Install focuspang on the devices that students will be using.

The student version of focuspang can be used on various devices.

From the left sidebar or the options below, select the device students will use:

- [MS Windows](/en/docs/middlehigh/student-guide/install-window)
- [Android](/en/docs/middlehigh/student-guide/install-android)
- [Chromebook](/en/docs/middlehigh/student-guide/install-chromebook)
- [iPad](/en/docs/middlehigh/student-guide/install-ipad)
