### Step 3: Install the App

Once you find the app, tap the Install button.

## After Installation

### Step 1: Launch the App

Open the focuspang Student app.

### Step 2: Allow Permissions

**1, Notification Permission**

<Image src="/img/en/middlehigh/student/12-001.jpg" size="lg" />

A pop-up will appear asking for notification access. Choose **Allow** so you can receive service notifications properly.

**2. Disable Battery Optimization**

<Image src="/img/en/middlehigh/student/12-02.jpg" size="lg" />

**Allow** this setting so the app can run in the background and maintain stable notifications.

**3. Camera/Video Recording Permission**

<Image src="/img/en/middlehigh/student/12-03.jpg" size="lg" />

A pop-up will ask for access to the camera and video recording. Select **Allow while using the app** to enable proper functionality.

### Step 3: Set Required Permissions

**1. Display Over Other Apps Permission**

<Image src="/img/en/middlehigh/student/12-04.jpg" size="lg" />

Tap **Set up** to open the Android permission settings.

<Image src="/img/en/middlehigh/student/12-05.jpg" size="lg" />

In the system screen, allow the focuspang app to display over other apps.

<Image src="/img/en/middlehigh/student/12-04.jpg" size="lg" />

Return to the setup screen and tap Next to proceed.

**2. Usage Access Permission**

<Image src="/img/en/middlehigh/student/12-06.jpg" size="lg" />

Tap **Set up** to go to the usage access settings.

<Image src="/img/en/middlehigh/student/12-07.jpg" size="lg" />

<Image src="/img/en/middlehigh/student/12-06.jpg" size="lg" />

Return to the setup screen and tap Finish to complete the process.

### Step 4: Log in with Invitation Code

<Image src="/img/en/middlehigh/student/12-08.jpg" size="lg" />

After completing all permissions, the login screen will appear. Enter the **invitation code** given by your teacher to log in.
