Click the **Download** above to get the Focuspang installation file.

### Step 2: Run the file

1. Once the download is complete, open your **Downloads folder.**

2. Find the file **FocuspangStudentSetup.exe.**

3. Double-click the file to run it.

### Step 3. Install the program

1. When the installer starts, click the **Next** button.

2. Click the **Agree** button to accept the Terms of Service

3. When installation is complete, click the **Finish** button.

4. After the installation window closes, the focuspang app will launch automatically.
   If it does not start automatically, double-click the focuspang icon on your desktop to open it manually.

- focuspang works best on the Chrome browser.
- If Chrome is not installed, please watch the Chrome installation tutorial video([ðŸ”—Link](https://focuspang.com/_dn_.html#Google)) and complete the installation.
