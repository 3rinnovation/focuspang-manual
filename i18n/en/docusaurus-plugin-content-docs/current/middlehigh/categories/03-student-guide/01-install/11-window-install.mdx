Click the **Download** button to get the installation file.

### Step 2: Run the file

1. Once the download is complete, open your **Downloads folder.**

2. Find the file **FocuspangStudentSetup.exe.**

3. Double-click the file to run it.

### Step 3. Start Installation

1. When the installation program starts, click the **Next** button.

2. lease click the **Agree** button for the terms of service.

3. Wait until the installation is finished, and click the **Finish** button.

4. When the installation window closes, the focuspang app will run automatically.
   If it doesn't start automatically, please double-click the focuspang icon installed on your desktop to start using focuspang.

- focuspang works best on the Chrome browser.
- If Chrome is not installed, please watch the Chrome installation tutorial video([ðŸ”—Link](https://focuspang.com/_dn_.html#Google)) and complete the installation.
