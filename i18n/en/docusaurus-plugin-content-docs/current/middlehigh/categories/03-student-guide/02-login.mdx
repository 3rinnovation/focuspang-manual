# 2. Login

1. Click the **[Invitation Code Login]** button in the pop-up window that appears when you launch focuspang.

2. Enter the class invitation code you received from your teacher.

   :::info

   If you can't type in English, press the Korean/English conversion key to make sure it is entered correctly.

   :::

<Image src="/img/en/middlehigh/student/05-01.jpg" size="lg" />
<Image src="/img/en/middlehigh/student/05-02.jpg" size="lg" />

3. Enter your name and 5 digits of your class number along with the invitation code to log in. Note that the student ID must be exactly 5 digits. For example, if you are in Grade 1, Class 8, you can enter 10108 or 01108, depending on your school's format.

   <Image src="/img/en/middlehigh/student/05-03.jpg" size="lg" />
