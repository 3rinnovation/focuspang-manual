# Focus TIme

#### 📍 Main Page → Click the class in the timetable → Click [Focus Time] in the left sidebar

1. When the teacher starts a class, the teacher's class screen is immediately displayed to students.

- A timer showing the remaining time for the class is displayed.
- Click the 🖌️ brush icon to display the palette (color selection) / eraser / trash can icons in the upper left corner.
  - Click the 🎨 palette to select text colors.
  - Click the ◽️ eraser to erase only the desired parts.
  - Click the 🗑️ trash can to erase all written content.
- Click [Full Screen View] to view the teacher's shared screen in full screen mode.

<Image src="/img/en/middlehigh/student/09-01.jpg" size="lg" />

2. **[Available Apps and Websites]** shows the list of apps and websites that the teacher has allowed.

<Image src="/img/en/middlehigh/student/09-02.jpg" size="lg" />

3. **[Shared Links/Files List]** displays the links shared by the teacher, and clicking on them will take you directly to the corresponding webpage.
