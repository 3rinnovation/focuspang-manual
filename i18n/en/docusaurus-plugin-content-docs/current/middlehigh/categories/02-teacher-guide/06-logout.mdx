# 6. Logout

1. On the home screen, click your profile picture or name in the upper left corner to go to the settings page.

2. Click the **[Logout]** button to log out.

:::danger Caution

- Please **make sure to log out** after class.
- Failure to log out may result in **duplicate login sessions**, in which case the student monitoring feature **may not work properly**.

:::
