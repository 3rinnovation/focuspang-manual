4. After creating a quiz, you'll automatically be taken to the Quizpang main page to manage your quiz. Click the **Start Quiz** button to allow students to take the quiz.

{" "}
<Image src="/img/en/middlehigh/teacher/05-04.jpg" size="lg" />

:::DANGER Caution

- You can't play QuizPang during FocusTime (class), so **make sure to end the class before starting QuizPang**.
- When the quiz is over, the teacher must click the **[End]** button to allow students to exit the quiz window.

:::
