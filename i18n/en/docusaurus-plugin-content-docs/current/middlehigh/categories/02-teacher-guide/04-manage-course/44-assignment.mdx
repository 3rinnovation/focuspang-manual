# Assignment Management

#### 📍 Main Page → Click the class in the timetable → Click [Assignment Management] in the left sidebar

### Create

1. Click the **[+Add Assignment]** button to go to the assignment creation screen.

<Image src="/img/en/middlehigh/teacher/06-01.jpg" size="lg" />

2. Create a title, assignment availability period, and content, then create the assignment.

<Image src="/img/en/middlehigh/teacher/06-02.jpg" size="lg" />

### Edit and Delete

You can edit a registered assignment by clicking [More] in the [Manage assignments] list, or delete an assignment by clicking the [Delete] button.

<Image src="/img/en/middlehigh/teacher/06-03.jpg" size="lg" />
