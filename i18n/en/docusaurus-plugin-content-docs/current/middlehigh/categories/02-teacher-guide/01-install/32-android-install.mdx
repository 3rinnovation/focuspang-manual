# Install focuspang on Android Device

## What we're going to do

We're going to install focuspang for teachers on your Android tablet or phone.
focuspang is a tool that helps you teach more effectively by letting you see student screens, give quizzes, and check attention levels.

## Before we start

Make sure you have:

- Your Android tablet or phone
- Internet connection
- About 5 minutes of time

## Step 1: Open Play Store

1. Find the **Play Store** app on your Android device
2. **Tap** the Play Store app to open it

## Step 2: Download focuspang Teacher App

Tap the button above or search for **focuspang Teacher** in the Play Store.

### Step 3: Install the app

Once you find the focuspang Teacher app, tap Install to begin installation.

## After Installation

### Step 1: Open the focuspang App

When installation is complete, you’ll see the focuspang Teacher app icon on your home screen. Tap the icon to start using focuspang.

### Step2: Allow each permission

**1. Allow the Display over other apps**

<Image src="/img/en/elementary/teacher/12-01.jpg" size="lg" />

When the app is first launched, you must **allow** Focuspang to display over other apps.

**2. Allow Notifications**

<Image src="/img/en/elementary/teacher/12-02.jpg" size="lg" />

To receive service notifications, select **Allow** when the popup appears.

**3. Exclude from Battery Optimization**

<Image src="/img/en/elementary/teacher/12-03.jpg" size="lg" />

To keep background execution and notifications active, **allow battery optimization exclusion.**

**4. Access the Login Screen**

<Image src="/img/en/elementary/teacher/12-04.jpg" size="lg" />

After granting all permissions, you can proceed to the **login screen.**
