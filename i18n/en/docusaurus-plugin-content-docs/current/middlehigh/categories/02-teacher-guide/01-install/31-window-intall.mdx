Click the **Download button** to get the installation file, or search for **focuspang Teacher** in the Play Store.

### Step 2: Run the file

1. Once the download is complete, open your **Downloads folder.**

2. Find the file **FocuspangTeacherSetup.exe.**

3. Double-click the file to run it.

### Step 3: Install the program

1. When the installation program starts, click the Next button.

2. Wait until the installation is finished.

3. Click the Finish button to complete the process.

## After Installation

Once installed, a focuspang Teacher app icon will appear on your desktop.

Double-click the icon to start using focuspang.
