## Helpful Information When Making an Inquiry

Providing the following details will help us give you faster and more accurate support:

### Basic Information

- **School name**: Which school you are using focuspang at
- **Administrator ID**: Account information related to the issue
- **Usage environment**: Browser type, operating system, etc.

### Problem Details

- **Time of occurrence**: When the issue started
- **Specific symptoms**: Detailed description of the issue
- **Error message**: Full content of any error messages shown
- **Steps to reproduce**: The process leading to the issue

### Additional Details

- **Screenshots**: Images showing the problem
- **Browser information**: Browser name and version
- **Actions taken**: Solutions you have already tried
