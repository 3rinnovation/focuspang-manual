# 1. Admin Role

- You can approve, edit, suspend, and delete **teacher accounts**.
- You can add other teachers as administrators and grant them **administrator privileges**.

:::warning Note

Administrators **focus on managing accounts and student information** and ddo not have access to class-related functionality.

:::
