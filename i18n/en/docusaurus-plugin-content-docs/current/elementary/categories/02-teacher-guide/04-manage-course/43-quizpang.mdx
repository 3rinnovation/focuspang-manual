# Quizpang

#### üìç Main Page ‚Üí Click the class in the timetable ‚Üí Click [Quizpang] in the left sidebar

1. Click the **[+Create Quiz]** button.

<Image src="/img/en/elementary/teacher/06-01.jpg" size="lg" />

2. Select [Quiz] from the Create Quiz pop-up window.

<Image src="/img/en/elementary/teacher/06-02.jpg" size="lg" />

3. Create a quiz name, question type, questions, passages, views, time limit, question commentary (optional), category (optional), and save.

<Image src="/img/en/elementary/teacher/06-03.jpg" size="lg" />
<Image src="/img/en/elementary/teacher/06-05.jpg" size="lg" />

4. Specify the correct answer by clicking the **[Correct Answer]** button to the right of the correct item in each question's view.

<Image src="/img/en/elementary/teacher/06-04.jpg" size="lg" />

5. After creating a quiz, you'll automatically be taken to the home screen, the main page, to manage your QuizPang. Click the **Start Quiz** button to allow students to take the quiz.

<Image src="/img/en/elementary/teacher/06-06.jpg" size="lg" />

6. Once the quiz has started, the quiz will appear on the student screen, and students will have a time limit set by you to complete the questions.

:::DANGER Caution.

- You can't play QuizPang during FocusTime (class), so **make sure to end the class before starting QuizPang**.
- When the quiz is over, the teacher must click the **[End]** button to allow students to exit the quiz window.

:::
