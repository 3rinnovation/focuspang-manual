# Group Activities

#### üìç Main Page ‚Üí Click the class in the timetable ‚Üí Click [Group Activities] in the left sidebar

1. Click the **[+Create New Moderated Activity Topic]** button to open the Create Group Topic pop-up window.

<Image src="/img/en/elementary/teacher/09-01.jpg" size="lg" />

2. Enter a group activity topic, specify the number of people per group, and click **[Randomize]** to automatically organize the group.

   :::info

   - If you want to reassign only specific students to different groups, you can do so by changing the group.
   - If you want to manually assign the groups, you can change the group for each student after the randomization.

   :::

<Image src="/img/en/elementary/teacher/09-02.jpg" size="lg" />

3. Once the groups are created, you can click the **[View groups Details]** button to see the list of students in each group.

4. Students in each group will be grouped together in Focus Time, and you can monitor each group separately.
