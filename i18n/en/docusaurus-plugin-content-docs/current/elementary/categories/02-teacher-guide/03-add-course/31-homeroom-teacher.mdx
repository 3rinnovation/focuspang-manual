# For homeroom teachers

## Creatinf a Class

### Step 1: Choose your grade and class

1. First, select your **grade**
2. Next, select your **class number**
3. If your class name is not a number (like "Love Class" or "Hope Class"), check the **Enter your own** box and type the class name

<Image src="/img/en/elementary/teacher/02-01.jpg" size="lg" />

### Step 2: Confirm your class

After creating your class, check the **top-left menu** to make sure it appears there.

<Image src="/img/en/elementary/teacher/02-02.jpg" size="lg" />

### Step 3: Invite students to your class

After creating your class or lessons, an **invitation code** is automatically created.
Give this invitation code to your students so they can join your lessons.

:::tip What is an invitation code?

- An invitation code is like a **password** that students need to enter your lessons
- Students enter the **invitation code + student ID + name** to join
- For homeroom teachers, all lessons for the same class use the **same invitation code**

:::

<Image src="/img/en/elementary/teacher/02-05.jpg" size="lg" />

## How to add lessons

### Step 1: Add a new lesson

1. Click the **[+ Add Lesson]** button on the timetable screen

<Image src="/img/en/elementary/teacher/03-01.jpg" size="lg" />

### Step 2: Fill in lesson details

2. In the pop-up window, enter the following information:
   - **Course name**: The subject you're teaching
   - **Grade**: Which grade level
   - **Class**: Which class number
   - **Day**: What day of the week
   - **Period**: What time period

### Step 3: Create the lesson

3. Click the **[Add Lesson]** button to create your lesson

:::info Adding multiple lessons
If you want to add the same lesson for other days of the week, you can add multiple lesson periods at once by clicking the **[+ Add Class Period]** button.
:::

<Image src="/img/en/elementary/teacher/03-02.jpg" size="lg" />

## How to edit and delete lessons

### Step 1: Select a lesson to edit

1. On the main timetable, click the **✏️pencil icon** in the upper right corner of the lesson you want to edit

<Image src="/img/en/elementary/teacher/03-03.jpg" size="lg" />

### Step 2: Make changes

2. After editing the lesson information:
   - Click the **[Save]** button to save your changes
   - Click the **[Delete]** button to remove the lesson completely

<Image src="/img/en/elementary/teacher/03-04.jpg" size="lg" />
