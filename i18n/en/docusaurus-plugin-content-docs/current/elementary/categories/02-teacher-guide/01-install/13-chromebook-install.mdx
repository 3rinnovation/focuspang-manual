# Using focuspang on Chromebook

:::info Note
Chromebooks can use focuspang directly in the web browser without installing any separate app.
:::

## How to use

### Step 1: Open Chrome browser

Open the **Chrome browser** on your Chromebook. (The Chrome browser comes pre-installed on all Chromebooks.)

### Step 2: Go to the focuspang Teacher site

Type [teacher.focuspang.com](https://teacher.focuspang.com) in the address bar or click the link to access the site.

### Step 3: Create a shortcut (optional)

For easier access, you can create a shortcut:

1. Open the focuspang site.
2. Click the **three-dot menu button** in the top right corner of Chrome browser.

<Image src="/img/en/elementary/teacher/00-01.png" size="lg" />

3. Click **Share, Save, and Share options**.

<Image src="/img/en/elementary/teacher/00-02.png" size="lg" />

4. Click **Create Shortcut**.

<Image src="/img/en/elementary/teacher/00-03.png" size="lg" />

5. Click **Create**.

<Image src="/img/en/elementary/teacher/00-04.png" size="lg" />

6. Click the **Start Manager ⦿** to confirm that the shortcut has been created.

<Image src="/img/en/elementary/teacher/00-05.png" size="lg" />

## Cautions

- Always use the Chrome browser.

- An internet connection is required.

- Allow microphone and camera permissions when prompted.

:::tip Help

- If you have difficulty using focuspang, please ask your school’s IT support teacher for assistance.
- Curious about Chromebook CEU settings? Check out the [**YouTube video**](https://www.youtube.com/watch?v=CLqLSEOcGyE&t=1247s).

:::
