# Install focuspang on Android Device

## What we're going to do

We’ll install the focuspang Teacher app on your Android tablet or phone.
focuspang helps you teach more effectively by letting you view student screens, give quizzes, and monitor attention levels.

## Before we start

Make sure you have:

- An Android tablet or phone
- An internet connection
- About 5 minutes of time

## Step 1: Open the Play Store

1. Find the **Play Store** app on your Android device.
2. **Tap** it to open.

## Step 2: Download the focuspang Teacher App

Click the button above or search for "**focuspang Teacher**" in the Play Store.

## Step 3: Install the app

1. Once you find the focuspang Teacher app, tap **Install**
2. Wait for the installation to finish
3. Tap **Open** to start using the app

## You're all done!

Once the installation is complete, you’ll see the Focuspang Teacher app icon on your home screen.
Tap it to open Focuspang and start managing your digital classroom.

:::info Important notice
Android mobile browsers (Chrome, Samsung Internet, etc.) may not work smoothly.
Please make sure to install and use the Teacher app for the best experience.
:::

:::tip Need help?
If you have trouble installing, please ask your school's IT support teacher for assistance.
:::

## What's next?

Now that you've installed focuspang, you can:

- [Log in to your teacher account](/docs/elementary/teacher-guide/login)
- Set up your first class
- Start using digital classroom features
