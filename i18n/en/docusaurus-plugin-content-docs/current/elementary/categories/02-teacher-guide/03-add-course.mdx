# 3. Creating Lessons

### Add a class

1. Click the **[+ Add class]** button on the timetable screen.

<Image src="/img/en/elementary/teacher/03-01.jpg" size="lg" />

2. In the pop-up window, enter **course name/grade/class/day/period**.

3. Click the **[Add Lesson]** button to create a lesson.

   :::info

   If you want to add the same class for other days of the week, you can add multiple classes at once by clicking the **[+ Add Class Period]** button.

   :::

<Image src="/img/en/elementary/teacher/03-02.jpg" size="lg" />

### Edit and Delete

1. Click the **✏️pencil icon** in the upper right corner of the lesson you want to edit on the main timetable.

<Image src="/img/en/elementary/teacher/03-03.jpg" size="lg" />

2. After editing the content, click the [Save] button to save changes, or click the [Delete] button to delete the lesson.

<Image src="/img/en/elementary/teacher/03-04.jpg" size="lg" />
