# Install focuspang on iPad

## Installation Steps

### Step 1: Open the App Store

On your iPad, find and open the **App Store.**

## Step 2: Search for FocusPang

In the search bar, type “focuspang” to find the app.

## Step 3: Install the App

Tap Install to download and install focuspang

## Step 4: Launch the App

Once installation is complete, tap the focuspang icon on the home screen to open it.

:::tip

If installation is difficult, please ask your school’s IT support teacher for assistance.

:::
