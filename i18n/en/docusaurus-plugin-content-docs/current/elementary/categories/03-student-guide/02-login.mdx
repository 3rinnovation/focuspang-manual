# 2. Login

1. Click the **[Invitation Code Login]** button in the pop-up window that appears after you launch FocusPan.

2. Enter the class invitation code you received from your teacher.

   :::info

   If you can't enter English, press the Korean/English conversion key to make sure it is entered correctly.

   :::

<Image src="/img/en/elementary/student/05-01.jpg" size="lg" />
<Image src="/img/en/elementary/student/05-02.jpg" size="lg" />

3. Enter your name and 5 digits of your class number along with the invitation code to log in. \*\*Please note that you can only enter 5 digits for your class number. For example, if you are a student in class 8 of year 1, you can enter 10108 or 01108, depending on your school's format.

   <Image src="/img/en/elementary/student/05-03.jpg" size="lg" />
