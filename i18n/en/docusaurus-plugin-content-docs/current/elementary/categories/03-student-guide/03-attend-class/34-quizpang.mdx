# Quizpang

#### 📍 Main Page → Click the class in the timetable → Click [Quizpang] in the left sidebar

1. When the teacher starts a quiz, the quiz begins immediately.

<Image src="/img/en/elementary/student/10-01.jpg" size="lg" />

2. When the time limit expires, students' rankings are automatically calculated based on their answers.

<Image src="/img/en/elementary/student/10-02.jpg" size="lg" />

3. Click the [Close] button to go to the quiz results page.

<Image src="/img/en/elementary/student/10-03.jpg" size="lg" />
