## Submitting Assignments

### Submit an Assignment

In the Submit Assignment tab, enter the title and content, then click Submit to turn in your assignment.

<Image src="/img/en/elementary/student/32-04.png" size="lg" />

- You can attach **files** if needed.
- If the assignment is not yet complete, you can **save it temporarily**.

### Edit an Assignment

You can edit an assignment before the due date.

<Image src="/img/en/elementary/student/32-05.png" size="lg" />

- Click the **Edit** button to activate the input fields.
- After making changes, click **Submit** button to save and submit the updated assignment.
