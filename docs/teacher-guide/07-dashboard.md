# 2.6 과제관리

* **\[과제 관리]** 메뉴에서 등록한 과제를 쉽게 관리할 수 있습니다.

  ![](/img/tcher_2-6_01.jpg)
* 과제 관리에서 **\[+과제 추가]** 클릭한 후 제목, 과제 공개 일시, 내용을 입력하고 등록합니다.

  ![](/img/tcher_2-6_02.jpg)
* **\[과제 관리]** 리스트의 **\[더보기]** 를 통해 과제 내용을 수정할 수 있고, **\[삭제]** 버튼을 클릭하면 삭제할 수도 있습니다. 

  ![](/img/tcher_2-6_03.jpg)
