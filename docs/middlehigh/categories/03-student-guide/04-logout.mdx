# 4. 로그아웃

수업이 끝나거나 포커스팡 사용을 마쳤다면 반드시 로그아웃을 해주세요. 올바른 로그아웃은 개인정보 보호와 다음 수업의 원활한 진행을 위해 매우 중요합니다.

## 로그아웃이 중요한 이유

포커스팡에서 로그아웃은 단순히 앱을 종료하는 것 이상의 의미를 가집니다:

- **개인정보 보호**: 다른 사람이 여러분의 계정에 접근하는 것을 방지
- **보안 강화**: 무단 사용으로부터 계정을 안전하게 보호
- **시스템 안정성**: 중복 로그인으로 인한 시스템 오류 방지
- **정확한 출석**: 수업 참여 기록이 정확하게 관리됨

## 로그아웃 방법

### 로그아웃 절차

홈 화면에서 **좌측 상단의 프로필 사진 또는 이름**을 클릭하면 설정 페이지로 이동합니다. 설정 페이지에서 **[로그아웃]** 버튼을 클릭하면 로그아웃이 완료됩니다.

## 로그아웃해야 하는 상황

### 필수 로그아웃 상황

다음 상황에서는 반드시 로그아웃해주세요:

#### 수업 종료 후

- **매 수업 종료 시**: 수업이 끝나면 즉시 로그아웃
- **쉬는 시간**: 긴 쉬는 시간이나 점심시간 전
- **하교 전**: 학교에서 사용 후 집에 가기 전

#### 기기 공유 시

- **컴퓨터실 사용 후**: 학교 컴퓨터실에서 사용 후
- **친구 기기 사용 후**: 친구의 기기를 빌려 사용한 후
- **공용 기기 사용 후**: 도서관이나 기타 공용 공간에서 사용 후

#### 보안이 필요한 경우

- **자리를 비울 때**: 잠시라도 자리를 비우게 될 때
- **다른 사람이 접근 가능한 곳**: 다른 사람이 기기에 접근할 수 있는 환경

## 로그아웃하지 않을 때 생기는 문제

### 개인정보 관련 문제

- **무단 접근**: 다른 사람이 여러분의 계정으로 접속할 수 있음
- **정보 노출**: 개인 학습 기록이나 성적 정보가 노출될 위험
- **부정 사용**: 다른 사람이 여러분 이름으로 활동할 수 있음

### 시스템 관련 문제

- **중복 로그인**: 여러 기기에서 동시 접속으로 인한 오류
- **출석 오류**: 정확한 수업 참여 기록이 어려움
- **기능 제한**: 일부 기능이 정상적으로 작동하지 않을 수 있음

## 로그아웃 후 확인사항

### 완전한 종료 확인

- **앱 종료**: 포커스팡 앱을 완전히 종료
- **브라우저 탭 닫기**: 웹 브라우저 사용 시 관련 탭 모두 닫기
- **로그인 화면 확인**: 다시 접속 시 로그인 화면이 나타나는지 확인

### 다음 사용 준비

- **기기 정리**: 사용한 기기를 깔끔하게 정리
- **개인 파일 삭제**: 다운로드한 파일이나 임시 파일 삭제
- **다음 수업 준비**: 다음 수업에 필요한 준비물 확인

## 로그아웃 관련 문제 해결

### 로그아웃이 되지 않는 경우

1. **앱 강제 종료**: 앱을 완전히 종료하고 다시 실행
2. **기기 재시작**: 스마트폰이나 태블릿을 재시작
3. **네트워크 확인**: 인터넷 연결 상태 확인
4. **선생님 문의**: 문제가 지속되면 담당 선생님에게 도움 요청

### 중복 로그인 문제 발생 시

1. **모든 기기에서 로그아웃**: 사용했던 모든 기기에서 로그아웃
2. **잠시 대기**: 5-10분 정도 기다린 후 다시 로그인
3. **하나의 기기만 사용**: 한 번에 하나의 기기에서만 로그인
4. **선생님 도움 요청**: 문제가 해결되지 않으면 선생님께 문의

## 올바른 로그아웃 습관

### 일상적인 습관

- **수업 후 즉시**: 수업이 끝나면 바로 로그아웃하는 습관
- **자리 비우기 전**: 잠시라도 자리를 비우기 전에 로그아웃
- **하루 마무리**: 하루 학습을 마치면 완전히 로그아웃

### 보안 의식

- **개인정보 보호**: 내 정보는 내가 지킨다는 의식
- **책임감**: 계정 사용에 대한 책임감 가지기
- **주의 깊은 사용**: 항상 주의 깊게 사용하고 마무리하기

### 도움 요청

- **모르는 것은 질문**: 확실하지 않은 것은 선생님께 질문
- **문제 즉시 신고**: 이상한 점이 발견되면 즉시 신고
- **경험 공유**: 친구들과 올바른 사용법 공유

올바른 로그아웃 습관을 통해 안전하고 효과적인 디지털 학습 환경을 만들어가세요!
