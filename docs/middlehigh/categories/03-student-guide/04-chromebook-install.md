# 4. 크롬북(Chromebook) 설치

- 크롬 확장 프로그램을 설치합니다. https://focuspang.com/\_dn\_.html#Google
- 크롬 브라우저에서 https://student.focuspang.com 사이트에 접속해 사용합니다.
- 선생님이 CEU를 이용하여 크롬 확장 프로그램을 모든 학생에게 강제 설치하고 학생이 삭제하지 못하도록 고정합니다.
