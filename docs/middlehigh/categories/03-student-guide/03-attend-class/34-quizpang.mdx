# 퀴즈팡

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [퀴즈팡] 클릭

## 퀴즈팡이 뭐예요?

퀴즈팡은 배운 내용을 문제로 풀어보는 재미있는 게임이에요.

퀴즈팡에서는

- 선생님이 수업 내용으로 문제를 만들어주세요.
- 친구들과 누가 더 많이 맞히는지 경쟁할 수 있어요.
- 내 점수를 확인하고 틀린 문제를 다시 공부할 수 있어요.

## 참여하는 방법

### 1단계: 퀴즈팡 참여하기

선생님이 퀴즈를 시작하면 카운트다운 후에 퀴즈가 시작됩니다.

### 2단계: 문제 풀기

제한 시간 내에 문제를 풀어보세요.

<Image src="/img/kr/middlehigh/student/10-01.jpg" size="lg" />

### 3단계: 결과 확인하기

제한 시간 종료 후 나의 퀴즈 랭킹을 확인할 수 있습니다.
