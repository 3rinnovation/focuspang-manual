# 3. 수업 참여

로그인을 완료했다면 이제 본격적으로 다양한 수업 활동에 참여해보세요! 포커스팡을 통해 더욱 효과적이고 흥미로운 학습 경험을 할 수 있습니다.

## 수업 참여란?

수업 참여는 선생님이 준비한 디지털 학습 활동에 능동적으로 참여하는 것을 의미합니다. 포커스팡에서는 다음과 같은 다양한 활동을 통해 학습할 수 있습니다:

- **수업 추가**: 각 과목별 수업을 시간표에 추가하여 체계적으로 관리
- **공지사항 확인**: 선생님의 중요한 안내사항과 수업 정보 확인
- **과제 제출**: 주어진 과제를 온라인으로 제출하고 피드백 받기
- **포커스타임**: 집중 학습 시간을 통해 효율적인 학습 환경 조성
- **퀴즈팡**: 재미있는 퀴즈를 통해 학습 내용 확인 및 복습
- **모둠 활동**: 동료들과 협력하여 프로젝트 및 토론 활동 참여

## 수업 참여의 특징

### 개별 수업 코드 시스템

각 수업마다 고유한 초대코드가 있어 수업별로 별도의 참여 절차가 필요합니다:

- **과목별 구분**: 수학, 영어, 과학 등 과목마다 다른 초대코드
- **선생님별 구분**: 같은 과목이라도 선생님이 다르면 다른 코드
- **학급별 구분**: 같은 선생님이라도 학급이 다르면 다른 코드

### 시간표 기반 관리

체계적인 학습 관리를 위해 시간표를 통해 수업을 관리합니다:

- **수업 추가**: 각 과목을 시간표에 추가하여 한눈에 확인
- **일정 관리**: 수업 시간과 과제 마감일 등을 체계적으로 관리
- **진도 확인**: 각 과목별 학습 진도와 참여도 확인

## 수업 참여 준비사항

### 기술적 준비

- **안정적인 인터넷 연결**: 원활한 수업 참여를 위한 네트워크 환경
- **기기 충전**: 수업 시간 동안 사용할 기기의 충전 상태 확인
- **앱 업데이트**: 최신 버전의 포커스팡 앱 사용

### 학습 환경 준비

- **조용한 공간**: 집중할 수 있는 학습 환경 조성
- **필요한 도구**: 필기구, 교재 등 수업에 필요한 준비물
- **시간 관리**: 수업 시작 전 미리 준비하여 시간 여유 확보

### 마음가짐 준비

- **적극적 참여**: 수업 활동에 능동적으로 참여하는 자세
- **집중력 유지**: 디지털 환경에서도 집중력을 유지하는 노력
- **소통 의지**: 선생님 및 동료들과 적극적으로 소통하는 자세

## 효과적인 수업 참여 방법

### 수업 전 준비

- **시간표 확인**: 오늘의 수업 일정과 준비사항 확인
- **과제 점검**: 제출해야 할 과제나 준비물 확인
- **복습**: 이전 수업 내용 간단히 복습

### 수업 중 참여

- **집중 유지**: 선생님의 설명에 집중하며 적극적으로 참여
- **질문하기**: 모르는 내용이 있으면 적극적으로 질문
- **상호작용**: 퀴즈나 토론 활동에 적극적으로 참여

### 수업 후 정리

- **내용 정리**: 배운 내용을 정리하고 복습
- **과제 확인**: 새로 주어진 과제나 공지사항 확인
- **피드백 활용**: 받은 피드백을 다음 학습에 활용

이제 각 활동별로 자세한 참여 방법을 알아보겠습니다!
