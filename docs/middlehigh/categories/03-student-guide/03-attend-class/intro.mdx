# 3. 수업 참여

로그인을 완료했다면 이제 본격적으로 다양한 수업 활동에 참여해보세요!

## 시간표 추가하기

### 수업 들어가기

초대코드로 로그인하면 수업이 자동으로 시간표에 나타납니다. 시간표에서 참여하고 싶은 수업을 클릭하면 수업에 참여할 수 있습니다.

### 수업 추가하기

로그인 이후, 새로운 과목 수업에 참여하고 싶을 때는 수업을 추가하면 됩니다. 메인페이지 시간표 옆에 있는 **[+수업추가]** 버튼을 눌러주세요

## 수업 활동

포커스팡에서는 다음과 같은 다양한 활동을 통해 학습할 수 있습니다.

- **수업 추가**: 각 과목별 수업을 시간표에 추가하여 체계적으로 관리
- **공지사항 확인**: 선생님의 중요한 안내사항과 수업 정보 확인
- **과제 제출**: 주어진 과제를 온라인으로 제출하고 피드백 받기
- **포커스타임**: 집중 학습 시간을 통해 효율적인 학습 환경 조성
- **퀴즈팡**: 재미있는 퀴즈를 통해 학습 내용 확인 및 복습
- **모둠 활동**: 동료들과 협력하여 프로젝트 및 토론 활동 참여

이제 각 활동별로 자세한 참여 방법을 알아보겠습니다!
