# 포커스타임

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [포커스타임] 클릭

1. 선생님 수업을 시작하면, 선생님의 수업화면이 학생들에게 바로 보여집니다.

- 해당 수업의 남은 시간을 나타내는 **타이머**가 표시됩니다.

- 🖌️붓 아이콘을 클릭하면, 왼쪽 상단에 팔레트(색상 선택) / 지우개/ 휴지통 아이콘이 나타납니다.
  - 🎨**팔레트**를 클릭하면 글씨 색을 선택할 수 있습니다.
  - ◽️**지우개**를 클릭하면 원하는 부분만 지울 수 있습니다.
  - 🗑️**휴지통**을 클릭하면 작성한 내용을 모두 지웁니다.
- **[전체화면보기]** 를 클릭하면 선생님이 공유한 화면을 전체 화면으로 볼 수 있습니다.

<Image src="/img/kr/middlehigh/student/09-01.jpg" size="lg" />

2. **[사용 가능한 앱과 웹사이트]**에서는 선생님이 허용한 앱과 웹사이트 목록을 확인할 수 있습니다.

<Image src="/img/kr/middlehigh/student/09-02.jpg" size="lg" />

3. **[공유된 링크/파일 목록]**에서는 선생님이 공유한 링크들이 표시되며, 클릭하면 해당 웹페이지로 바로 이동할 수 있습니다.

<Image src="/img/kr/middlehigh/student/09-03.jpg" size="lg" />
