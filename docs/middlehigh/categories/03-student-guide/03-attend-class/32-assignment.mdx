# 과제

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [과제] 클릭

## 과제 확인하기

### 1단계: 과제 목록

과제 목록에서 제출 현황, 제출 기한을 확인할 수 있습니다.

<Image src="/img/kr/middlehigh/student/32-01.png" size="lg" />

### 2단계: 과제 정보 확인

제출할 과제를 클릭하여 과제 내용 및 첨부파일을 확인합니다.

<Image src="/img/kr/middlehigh/student/32-02.png" size="lg" />
<Image src="/img/kr/middlehigh/student/32-03.png" size="lg" />

## 과제 제출하기

### 과제 제출

**과제 제출** 탭에서 제목, 내용을 작성하고 **제출하기** 버튼을 클릭하여 과제를 제출합니다.

<Image src="/img/kr/middlehigh/student/32-04.png" size="lg" />

- **첨부 파일**을 추가할 수 있습니다.
- 과제가 아직 미완성일 경우 **임시저장**할 수 있습니다.

### 과제 수정

아직 마감 기한이 지나지 않은 과제에 한해서 과제 내용을 수정할 수 있습니다.

<Image src="/img/kr/middlehigh/student/32-05.png" size="lg" />

- **수정하기** 버튼을 누르면 입력창이 활성화 됩니다.
- 수정 완료 후 **제출하기** 버튼을 눌러야 변경된 사항으로 과제가 제출됩니다.

## 점수 확인하기

### 1단계: 과제 목록

과제 목록에서 점수를 확인할 과제를 클릭합니다.

<Image src="/img/kr/middlehigh/student/32-06.png" size="lg" />

### 2단계: 과제 정보 확인

**과제 정보** 탭 하단에서 점수와 선생님의 코멘트를 확인할 수 있습니다.

<Image src="/img/kr/middlehigh/student/32-07.png" size="lg" />
