# 퀴즈팡

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [퀴즈팡] 클릭

1. 선생님이 퀴즈를 시작하면 카운트다운 후에 퀴즈가 시작됩니다.

<Image src="/img/kr/middlehigh/student/10-01.jpg" size="lg" />

2. 문제를 풀고 문제 응답 제한 시간이 지나면 자동으로 퀴즈 결과 페이지로 이동합니다.

- 퀴즈 결과 페이지의 **[랭킹보기]** 를 통해 나의 퀴즈 랭킹을 확인할 수 있습니다.

  <Image src="/img/kr/middlehigh/student/10-02.jpg" size="lg" />
