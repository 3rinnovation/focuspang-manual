# 수업 추가

#### 📍 메인 페이지 → 시간표에서 클릭

1. 메인페이지 시간표 옆 **[+수업추가]** 를 클릭합니다.

<Image src="/img/kr/middlehigh/student/06-01.jpg" size="lg" />

:::danger 주의

수업 중에는 메인페이지로 이동할 수 없으니 수업이 끝난 후 **[처음으로]** 아이콘 클릭해서 이동합니다.

:::

2. 추가할 수업의 초대코드를 입력후 **[수업 추가]** 버튼을 클릭합니다.

<Image src="/img/kr/middlehigh/student/06-02.jpg" size="lg" />

:::info 기기별 상세

- **안드로이드**: 선생님이 수업을 시작하면 화면 녹화 권한 요청 팝업이 나타납니다. **[시작하기]** 버튼을 클릭하여 화면 공유를 허용합니다.
- **크롬북:** 선생님이 수업을 시작하면 화면 공유 정보 선택 팝업이 나타납니다. 이 팝업에서 **전체 화면** 옵션을 선택한 후, **[공유]** 버튼을 클릭하여 화면을 공유합니다.
- **윈도우**: 선생님이 수업을 시작하면 별도의 학생 작업 없이 자동으로 화면 공유가 진행됩니다.

:::
