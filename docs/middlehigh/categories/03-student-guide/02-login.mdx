# 2. 로그인

1. 포커스팡을 실행한 후 실행되는 팝업 창에서 **[초대코드 로그인]** 버튼을 클릭합니다.

2. 선생님에게 받은 **수업 초대코드를 입력**합니다.

   :::info

   영문 입력이 되지 않으면, 한/영 변환키를 눌러 영문으로 정확히 입력되는지 확인해 주세요.

   :::

<Image src="/img/kr/middlehigh/student/05-01.jpg" size="lg" />
<Image src="/img/kr/middlehigh/student/05-02.jpg" size="lg" />

3. 초대코드와 함께 **학번 5자리**와 **이름**을 입력한 뒤 로그인합니다.

:::tip 학번 입력하는 방법
학번은 **숫자 5개**만 입력해 주세요.
예를 들어, 1학년 1반 8번 학생이라면:

- 10108 또는 01108처럼 입력하면 돼요
- 학교에서 알려준 학번 형식에 맞게 입력해 주세요
  :::
