# 2. 로그인

포커스팡 앱을 설치했다면 이제 선생님의 수업에 참여해보세요! 로그인을 통해 디지털 학습 환경에서 다양한 교육 활동에 참여할 수 있습니다.

## 로그인이란?

로그인은 여러분이 선생님의 디지털 수업에 참여하는 과정입니다. 온라인 교실에 입장하는 것과 같은 개념으로, 로그인을 완료하면:

- **수업 참여**: 선생님이 준비한 다양한 수업 활동에 참여
- **학습 자료 접근**: 수업 자료, 과제, 공지사항 등 확인
- **실시간 소통**: 선생님 및 다른 학생들과 실시간 소통
- **학습 관리**: 개인 학습 진도 및 성과 관리

## 로그인 절차

### 1단계: 초대코드 로그인 선택

포커스팡을 실행한 후 나타나는 팝업 창에서 **[초대코드 로그인]** 버튼을 클릭합니다.

<Image src="/img/kr/middlehigh/student/05-01.jpg" size="lg" />

### 2단계: 초대코드 입력

선생님에게 받은 **수업 초대코드를 정확히 입력**합니다.

:::tip 초대코드 입력 시 주의사항

- 초대코드는 선생님이 제공하는 고유한 수업 식별 코드입니다
- 대소문자를 구분하여 정확히 입력해야 합니다
- 영문 입력이 되지 않으면 한/영 변환키(한글/영어)를 눌러 영문으로 전환하세요
- 코드를 복사해서 붙여넣기 할 때는 앞뒤 공백이 없는지 확인하세요
  :::

<Image src="/img/kr/middlehigh/student/05-02.jpg" size="lg" />

### 3단계: 개인정보 입력

초대코드와 함께 **학번 5자리**와 **이름**을 정확히 입력한 후 로그인합니다.

:::tip 학번 입력 가이드
**학번 형식**: 숫자 5자리로 입력

- 예시: 1학년 1반 8번 학생의 경우
  - 10108 또는 01108 형태로 입력
  - 학교에서 안내한 학번 형식을 따라 입력
- **이름**: 학교에 등록된 정식 이름으로 입력
- **정확성**: 잘못된 정보 입력 시 수업 참여에 문제가 발생할 수 있음
  :::

<Image src="/img/kr/middlehigh/student/05-03.jpg" size="lg" />

## 로그인 문제 해결

### 자주 발생하는 문제들

#### 초대코드 관련 문제

- **코드 오류**: 선생님에게 정확한 초대코드를 다시 확인
- **입력 오류**: 대소문자, 숫자, 특수문자를 정확히 구분하여 입력
- **만료된 코드**: 선생님께 새로운 초대코드 요청

#### 개인정보 관련 문제

- **학번 오류**: 학교에서 안내한 정확한 학번 형식으로 입력
- **이름 불일치**: 학교에 등록된 정식 이름과 일치하는지 확인
- **중복 계정**: 이미 다른 기기에서 로그인한 경우 기존 세션 종료

#### 기술적 문제

- **네트워크 오류**: Wi-Fi 연결 상태 확인
- **앱 오류**: 앱을 완전히 종료 후 재시작
- **기기 문제**: 다른 기기에서 로그인 시도

### 해결 방법

1. **정보 재확인**: 입력한 모든 정보가 정확한지 다시 확인
2. **앱 재시작**: 포커스팡 앱을 완전히 종료하고 다시 실행
3. **네트워크 확인**: 안정적인 인터넷 연결 확인
4. **선생님 문의**: 문제가 지속되면 담당 선생님에게 도움 요청

## 로그인 성공 후 할 일

### 첫 로그인 시 확인사항

- **프로필 설정**: 개인 정보가 정확히 표시되는지 확인
- **수업 목록**: 참여할 수업들이 올바르게 표시되는지 확인
- **기능 탐색**: 주요 기능들을 미리 둘러보기

### 수업 참여 준비

- **수업 일정**: 시간표를 확인하여 수업 일정 파악
- **알림 설정**: 중요한 공지사항을 놓치지 않도록 알림 설정
- **학습 환경**: 집중할 수 있는 학습 환경 조성

## 보안 및 개인정보 보호

### 계정 보안

- **비밀번호**: 다른 사람과 계정 정보 공유 금지
- **로그아웃**: 사용 후 반드시 로그아웃
- **개인정보**: 개인정보를 함부로 공개하지 않기

### 올바른 사용

- **수업 목적**: 학습 목적으로만 사용
- **규칙 준수**: 학교 및 수업 규칙 준수
- **존중**: 선생님과 다른 학생들에 대한 존중

로그인을 완료했다면 이제 포커스팡을 통해 더욱 효과적이고 흥미로운 디지털 학습을 경험해보세요!
