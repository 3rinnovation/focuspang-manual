# 윈도우 기기에 포커스팡 설치하기

## 설치 방법

### 1단계: 설치 파일 받기

<div class="fp-download-container fp-download-blue">
  <div>
    <div class="fp-download-title">윈도우용 설치 파일</div>
    <div class="fp-download-desc">FocuspangStudentSetup.exe</div>
  </div>
  <a
    class="fp-download-button"
    href="https://obj-g-1.ktcloud.com/focuspang-media/downloads/FocuspangStudentSetup.exe"
  >
    <img
      class="fp-download-icon"
      src="/img/download-white-icon.svg"
      alt="다운로드"
    />
    다운로드
  </a>
</div>

위 **다운로드 버튼**을 눌러 포커스팡 설치 파일을 받아 주세요.

### 2단계: 파일 실행하기

1. 다운로드가 완료되면 **다운로드 폴더**를 열어주세요.
2. **FocuspangStudentSetup.exe** 파일을 찾아 더블클릭하여 실행해 주세요.

### 3단계: 설치하기

1. 설치 프로그램이 시작되면 **다음** 버튼을 눌러주세요.

   <Image src="/img/kr/middlehigh/student/11-01.png" size="md" />

2. 서비스 이용 약관에 대해 **동의함** 버튼을 눌러주세요.

   <Image src="/img/kr/middlehigh/student/11-02.png" size="md" />

3. 설치가 완료되고 **마침** 버튼을 눌러주세요.

   <Image src="/img/kr/middlehigh/student/11-03.png" size="md" />

4. 설치 창이 닫히면, 포커스팡앱이 자동으로 실행됩니다. 만약 자동으로 실행되지 않는다면 바탕화면에 설치된 포커스팡 아이콘을 더블 클릭하여 직접 열어주세요.

### 4단계: 크롬 익스텐션 설치[(🔗바로가기)](https://chromewebstore.google.com/detail/%ED%8F%AC%EC%BB%A4%EC%8A%A4%ED%8C%A1-ai-%EC%9C%88%EB%8F%84%EC%9A%B0%EC%9A%A9/nkcbfkbfngbaebldgenpafgffpgdhibi?hl=ko&utm_source=ext_sidebar)

1. 포커스팡 실행 시 크롬 익스텐션 설치 알람이 뜨면, **확인**을 눌러주세요.

   <Image src="/img/kr/middlehigh/student/11-04.png" size="sm" />

2. 크롬 웹스토어에서 포커스팡 ai를 **Chrome에 추가**해 주세요.

   <Image src="/img/kr/middlehigh/student/11-05.png" size="lg" />

3. **확장 프로그램 추가** 버튼을 눌러 설치를 완료해 주세요.

   <Image src="/img/kr/middlehigh/student/11-06.png" size="sm" />

:::tip 도움말

설치가 어려우시면 학교 컴퓨터 담당 선생님께 도움을 요청해 주세요.

:::
