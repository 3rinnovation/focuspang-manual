# 1. 관리자 역할

포커스팡 관리자는 학교에서 포커스팡을 효율적으로 운영하기 위해 선생님들의 계정과 권한을 관리하는 중요한 역할을 담당합니다.

## 관리자의 주요 역할

### 1. 계정 관리 업무

- **신규 계정 승인**: 새로 가입한 선생님 계정을 검토하고 승인
- **계정 정보 관리**: 선생님 정보 수정, 비밀번호 초기화
- **계정 상태 관리**: 휴직, 퇴직 등에 따른 계정 정지 및 삭제
- **학생 정보 관리**: 학생 이름, 학번 등 기본 정보 수정

### 2. 권한 관리 업무

- **관리자 권한 부여**: 다른 선생님에게 관리자 권한 위임
- **접근 권한 통제**: 승인되지 않은 사용자의 접근 차단
- **역할별 권한 설정**: 각 선생님의 담당 업무에 맞는 권한 부여

## 관리자 지정 기준

일반적으로 다음과 같은 선생님이 관리자가 됩니다.

- **정보부장 선생님**: 학교 IT 시스템 전반을 관리
- **컴퓨터 담당 선생님**: 디지털 교육 도구 운영 경험
- **교무부장 선생님**: 학교 행정 업무 총괄
- **연구부장 선생님**: 교육 혁신 및 새로운 도구 도입 담당

:::tip 관리자와 일반 선생님의 차이점

- **관리자**: 계정 관리, 권한 부여, 시스템 운영 담당
- **일반 선생님**: 수업 진행, 학생 관리, 교육 활동 담당

관리자도 일반 선생님처럼 수업을 진행할 수 있지만, 추가로 계정 관리 권한을 가집니다.
:::
