# 4. 사용 가이드

포커스팡 관리자 시스템을 처음 사용하시거나 더 자세한 사용법을 알고 싶으시다면 동영상 가이드를 확인해 보세요.

## 동영상 가이드

관리자 시스템 왼쪽 사이드 바 **사용 가이드** 메뉴 내 버튼을 클릭하시면 동영상으로 이동합니다.

<Image src="/img/kr/elementary/admin/04-01.jpg" size="lg" />

## 추가 지원 서비스

포커스팡 사용 중 궁금한 점이나 문제가 발생했을 때 즉시 도움을 받을 수 있습니다.

### 즉문즉답 [🔗바로가기](https://focuspang.com/ask.html)

- **24시간 접수**: 언제든지 문의 등록 가능
- **빠른 응답**: 영업시간 내 신속한 답변 제공
- **문의 이력**: 이전 문의 내역 확인 가능

### 카카오톡 문의 [🔗바로가기](https://pf.kakao.com/_xjxgGDxj)

- **실시간 상담**: 영업시간 내 실시간 채팅 상담
- **파일 전송**: 오류 화면 캡처 등 파일 전송 가능
- **상담 기록**: 카카오톡 대화 내역으로 상담 기록 보관

## 문의 시 도움이 되는 정보

문의할 때 다음 정보를 함께 제공하면 더 빠르고 정확한 답변을 받을 수 있습니다.

### 기본 정보

- **학교명**: 어느 학교에서 사용 중인지
- **관리자 아이디**: 문제가 발생한 계정 정보
- **사용 환경**: 브라우저 종류, 운영체제 등

### 문제 상황

- **발생 시점**: 언제부터 문제가 발생했는지
- **구체적 증상**: 어떤 문제가 발생하는지 상세히 설명
- **오류 메시지**: 나타나는 오류 메시지 전체 내용
- **재현 단계**: 문제가 발생하는 단계별 과정

### 부가 내용

- **화면 캡처**: 문제 상황의 스크린샷
- **브라우저 정보**: 사용 중인 브라우저와 버전
- **시도한 해결책**: 이미 시도해본 해결 방법들
