# 수업 공지

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [수업 공지] 클릭

### 등록

1. [+ 수업 공지] 버튼을 클릭하여 새로운 공지 작성 화면으로 이동합니다.

<Image src="/img/kr/middlehigh/teacher/03-01.jpg" size="lg" />

2. 제목, 공지 공개 일시, 내용을 입력하고 [등록] 버튼을 누르면 공지가 등록됩니다.

   :::info

   - **공지 공개 시점을 설정**할 수 있습니다.
     - 즉시: 공지를 등록하는 즉시 학생들에게 공개됩니다.
     - 예약: 원하는 날짜와 시간을 지정하여 공지를 예약 공개할 수 있습니다.
   - [+ 공지할 반 추가하기] 버튼을 클릭하면, 현재 작성 중인 내용을 여러 반에 **동시에 공지**할 수 있습니다.
   - **이미지, 사진, 첨부파일**도 함께 추가해 공지할 수도 있습니다.
   - 등록된 공지사항은 학생용 앱/웹에서 확인할 수 있습니다.

   :::

<Image src="/img/kr/middlehigh/teacher/03-02.jpg" size="lg" />

### 수정 및 삭제

1. 수정하려는 공지가 있다면, 공지 목록에서 해당 공지 우측의 **[내용 보기]** 버튼을 클릭하여 상세 화면으로 이동합니다.
2. 공지 목록에서 삭제하려는 공지의 우측에 있는 **[삭제]** 버튼을 클릭하여 공지를 삭제합니다.

<Image src="/img/kr/middlehigh/teacher/03-03.jpg" size="lg" />

:::tip

공지 상세 화면에서 해당 공지사항을 읽은 학생과 읽지 않은 학생의 목록을 확인할 수 있습니다.

:::
