# 학생 명단

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [학생 명단] 클릭

<Image src="/img/kr/middlehigh/teacher/08-01.jpg" size="lg" />

### 수정하기

✏️연필 아이콘을 클릭하면 학생의 학번과 이름을 수정할 수 있습니다.

<Image src="/img/kr/middlehigh/teacher/08-02.jpg" size="lg" />

### 내보내기

🚪문 모양 아이콘을 클릭하면 해당 학생을 명단에서 삭제할 수 있습니다.

<Image src="/img/kr/middlehigh/teacher/08-03.jpg" size="lg" />
