# 퀴즈팡

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [퀴즈팡] 클릭

1. **[+퀴즈 만들기]** 버튼을 클릭합니다.

<Image src="/img/kr/middlehigh/teacher/05-01.jpg" size="lg" />

2. 퀴즈 만들기 팝업 창에서 [퀴즈]를 선택합니다.

<Image src="/img/kr/middlehigh/teacher/05-02.jpg" size="lg" />

3. 퀴즈 이름, 문제 유형, 문제, 지문, 보기를 작성하고 제한 시간과 문제 해설(선택), 카테고리(선택)를 작성한 후 저장합니다.

<Image src="/img/kr/middlehigh/teacher/05-03.jpg" size="lg" />

4. 퀴즈를 만들면 자동으로 홈 화면인 메인 페이지의 퀴즈팡 관리로 넘어갑니다. **퀴즈 시작** 버튼을 클릭하면 학생들이 퀴즈를 풀 수 있습니다.

<Image src="/img/kr/middlehigh/teacher/05-04.jpg" size="lg" />

:::danger 주의

- 포커스타임(수업) 중에는 퀴즈팡을 동시에 할 수 없으니, **반드시 수업을 종료한 후 퀴즈팡을 시작해 주세요.**
- 퀴즈가 끝나면 선생님께서는 반드시 **[종료]** 버튼을 눌러야 학생들이 퀴즈 창에서 나올 수 있습니다.

:::
