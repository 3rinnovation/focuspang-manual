# 8. 학생 명단

- **[학생 명단]** 을 클릭하면 학생 명단 페이지로 이동합니다.

- 연필 아이콘을 클릭하면 학생의 반과 이름을 수정할 수 있습니다.
- 문모양 아이콘을 클릭하면 수업에 등록된 학생을 내보낼 수 있습니다.
