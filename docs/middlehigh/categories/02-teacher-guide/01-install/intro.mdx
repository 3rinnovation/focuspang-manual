# 1. 설치

안녕하세요!
선생님이 사용하실 기기에 맞는 포커스팡을 설치해보겠습니다.

포커스팡 선생님용은 윈도우 컴퓨터, 안드로이드 기기, 크롬북, 맥북에서 사용할 수 있습니다.

아래에서 선생님이 사용하실 기기를 선택해 주세요.

:::info 참고사항

- 윈도우나 맥북에서는 **크롬 브라우저로 teacher.focuspang.com에 접속**해서도 사용할 수 있어요
- 아이패드용 선생님 앱은 곧 출시될 예정입니다
- 안드로이드 모바일 브라우저로는 원활한 이용이 어려우니 앱을 설치해 주세요
  :::
