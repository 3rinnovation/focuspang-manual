# 윈도우 컴퓨터에 포커스팡 설치하기

:::info 참고

따로 앱을 설치하지 않고, **크롬 브라우저**에서 [**🔗홈페이지**](https://teacher.focuspang.com)에 접속하여 바로 사용할 수도 있습니다.

:::

## 설치 방법

### 1단계: 파일 다운로드하기

<div class="fp-download-container fp-download-blue">
  <div>
    <div class="fp-download-title">윈도우용 선생님 설치 파일</div>
    <div class="fp-download-desc">FocuspangTeacherSetup.exe</div>
  </div>
  <a
    class="fp-download-button"
    href="https://obj-g-1.ktcloud.com/focuspang-media/downloads/FocuspangTeacherSetup.exe"
  >
    <img
      class="fp-download-icon"
      src="/img/download-white-icon.svg"
      alt="다운로드"
    />
    다운로드
  </a>
</div>

위 다운로드 버튼을 눌러 포커스팡 설치 파일을 받아 주세요.

### 2단계: 파일 실행하기

1. 다운로드가 완료되면 **다운로드 폴더**를 열어주세요.
2. **FocuspangTeacherSetup.exe** 파일을 찾아주세요.
3. 파일을 더블클릭해서 실행해 주세요.

### 3단계: 설치하기

1. 설치 프로그램이 시작되면 **다음** 버튼을 눌러주세요.

   <Image src="/img/kr/elementary/teacher/11-001.png" size="md" />

2. 서비스 이용 약관에 대해 **동의함** 버튼을 눌러주세요.

   <Image src="/img/kr/elementary/teacher/11-02.png" size="md" />

3. 설치가 완료되고 **마침** 버튼을 눌러주세요.

   <Image src="/img/kr/elementary/teacher/11-03.png" size="md" />

4. 설치 창이 닫히면, 포커스팡앱이 자동으로 실행됩니다. 만약 자동으로 실행되지 않는다면 바탕화면에 설치된 포커스팡 아이콘을 더블 클릭하여 직접 열어주세요.

   <Image src="/img/kr/elementary/teacher/11-04.png" size="lg" />

:::tip 도움말
설치가 어려우시면 학교 컴퓨터 담당 선생님께 도움을 요청해 주세요.
:::
