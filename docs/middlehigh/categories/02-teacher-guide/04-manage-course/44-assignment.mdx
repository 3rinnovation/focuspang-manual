# 과제관리

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [과제 관리] 클릭

:::tip 과제 관리로 무엇을 할 수 있나요?

- 학생들에게 과제를 내줄 수 있어요
- 학생들이 과제를 제출했는지 확인할 수 있어요
- 제출된 과제 파일을 다운로드받을 수 있어요
- 과제 내용을 수정하거나 삭제할 수 있어요
  :::

### 등록

1. **[+ 과제 추가]** 버튼을 클릭하여 새로운 과제 등록 화면으로 이동합니다.

<Image src="/img/kr/middlehigh/teacher/06-01.jpg" size="lg" />

2. 제목, 과제 공개 일시, 내용을 작성한 후 과제를 등록합니다.

<Image src="/img/kr/middlehigh/teacher/06-02.jpg" size="lg" />

### 수정 및 삭제

등록된 과제는 **[과제 관리]** 리스트의 **[더보기]**를 클릭하여 내용을 수정하거나, **[삭제]** 버튼으로 과제를 삭제할 수 있습니다.

<Image src="/img/kr/middlehigh/teacher/06-03.jpg" size="lg" />
