# 4. 수업 진행

시간표에서 수업을 클릭하면 해당 수업의 관리 화면으로 이동합니다. 이 화면에서는 디지털 교육 환경에서 필요한 모든 수업 관리 기능을 체계적으로 활용할 수 있습니다.

## 수업 진행 화면 구성

수업 진행 화면은 효율적인 디지털 수업 운영을 위해 다음과 같은 기능들로 구성되어 있습니다:

### 핵심 수업 관리 기능

- **수업 공지**: 학생들에게 중요한 정보와 안내사항을 효과적으로 전달
- **포커스 타임**: 집중 학습 환경 조성 및 실시간 학습 모니터링
- **퀴즈팡**: 학습 평가 및 즉시 피드백을 통한 학습 효과 향상
- **과제 관리**: 체계적인 과제 출제, 수집, 평가 시스템
- **학생 관리**: 개별 학생 학습 현황 및 참여도 모니터링
- **모둠 활동**: 협력 학습 및 그룹 프로젝트 관리

## 수업 진행의 특징

### 통합 관리 시스템

하나의 화면에서 수업과 관련된 모든 활동을 통합적으로 관리할 수 있어 수업 진행의 효율성을 크게 향상시킵니다.

### 실시간 모니터링

학생들의 학습 활동을 실시간으로 모니터링하여 즉각적인 피드백과 지도가 가능합니다.

### 데이터 기반 교육

학생들의 참여도, 이해도, 학습 패턴 등을 데이터로 분석하여 보다 효과적인 교육 전략을 수립할 수 있습니다.

각 기능별 상세한 사용법은 해당 메뉴를 클릭하여 확인하실 수 있습니다.
