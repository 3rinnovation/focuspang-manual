# 6. 정보 변경

개인정보 보호와 효율적인 수업 운영을 위해 계정 정보를 최신 상태로 유지하는 것이 중요합니다. 정기적인 정보 업데이트를 통해 안전하고 효과적인 디지털 수업 환경을 구축하세요.

## 정보 변경의 중요성

선생님 계정의 정보를 정확하게 유지하는 것은 다음과 같은 이유로 중요합니다:

- **신원 확인**: 학생들이 선생님을 쉽게 식별할 수 있도록 지원
- **보안 강화**: 최신 연락처로 보안 알림 및 중요 공지 수신
- **소통 개선**: 정확한 정보로 학교 관리자 및 동료와 원활한 소통
- **시스템 안정성**: 정확한 정보로 시스템 오류 방지

## 설정 페이지 접속

### 설정 페이지 이동 방법

홈 화면에서 **좌측 상단의 프로필 사진 또는 이름**을 클릭하면 설정 페이지로 이동합니다.

## 프로필 사진 관리

### 프로필 사진의 역할

- **신원 확인**: 학생들이 선생님을 쉽게 식별
- **친근감 조성**: 학생들과의 유대감 형성
- **전문성 표현**: 교육자로서의 전문적 이미지 구축

### 프로필 사진 변경 방법

1. **추가/변경 버튼 클릭**: 프로필 사진 옆의 **[+] 버튼** 클릭
2. **이미지 선택**: 컴퓨터에서 적절한 사진 파일 선택
3. **크기 조정**: 필요시 사진 크기 및 위치 조정
4. **저장**: 변경사항 저장

### 프로필 사진 선택 가이드

- **얼굴이 명확히 보이는 사진**: 학생들이 쉽게 식별할 수 있도록
- **전문적인 이미지**: 교육자로서 적절한 복장과 표정
- **고화질 이미지**: 선명하고 깨끗한 이미지 사용
- **최신 사진**: 현재 모습과 유사한 사진 사용

## 개인정보 변경

### 변경 가능한 정보

- **이름**: 개명이나 호칭 변경 시
- **이메일**: 연락 가능한 최신 이메일 주소로 업데이트
- **전화번호**: 긴급 연락 시 사용할 번호
- **소속 정보**: 부서 이동이나 담당 과목 변경 시

### 정보 변경 절차

1. **수정할 정보 입력**: 변경하려는 이름 또는 이메일 입력
2. **정보 확인**: 입력한 정보가 정확한지 다시 한 번 확인
3. **변경 저장**: **[정보 변경]** 버튼 클릭하여 변경사항 저장
4. **확인**: 변경이 제대로 적용되었는지 확인

### 정보 변경 시 주의사항

- **정확성**: 모든 정보는 정확하게 입력
- **일관성**: 학교 시스템의 다른 정보와 일치하도록 유지
- **즉시 반영**: 변경사항은 즉시 시스템에 반영됨
- **동료 안내**: 필요시 동료 선생님들에게 변경사항 안내

## 비밀번호 변경

### 비밀번호 변경이 필요한 경우

- **정기 변경**: 보안을 위해 3-6개월마다 정기 변경
- **보안 우려**: 비밀번호 유출 의심 시
- **공유 후**: 다른 사람과 비밀번호를 공유한 후
- **시스템 권고**: 시스템에서 비밀번호 변경을 권고할 때

### 비밀번호 변경 절차

1. **현재 비밀번호 입력**: 보안 확인을 위해 현재 비밀번호 입력
2. **다음 단계**: **[다음]** 버튼 클릭하여 다음 단계로 이동
3. **새 비밀번호 입력**: 새로운 비밀번호 입력 및 확인
4. **변경 완료**: **[비밀번호 변경]** 버튼 클릭하여 변경 완료

### 안전한 비밀번호 생성 가이드

#### 비밀번호 구성 요소

- **길이**: 8자 이상 (12자 이상 권장)
- **복잡성**: 대문자, 소문자, 숫자, 특수문자 조합
- **고유성**: 다른 사이트와 다른 비밀번호 사용
- **예측 불가**: 개인정보와 관련 없는 내용

#### 피해야 할 비밀번호

- **개인정보 포함**: 이름, 생년월일, 전화번호 등
- **단순 패턴**: 123456, qwerty 등
- **사전 단어**: 일반적인 단어나 문구
- **이전 비밀번호**: 이전에 사용했던 비밀번호

#### 비밀번호 관리 팁

- **비밀번호 관리자 사용**: 안전한 비밀번호 생성 및 관리
- **정기적 변경**: 3-6개월마다 정기적으로 변경
- **메모 금지**: 비밀번호를 컴퓨터나 메모지에 기록하지 않기
- **공유 금지**: 다른 사람과 비밀번호 공유하지 않기

## 정보 변경 후 확인사항

### 변경사항 확인

- **정보 정확성**: 변경된 정보가 정확히 반영되었는지 확인
- **시스템 반영**: 다른 페이지에서도 변경사항이 적용되었는지 확인
- **기능 테스트**: 로그인 및 주요 기능이 정상 작동하는지 확인

### 관련 시스템 업데이트

- **이메일 확인**: 새 이메일 주소로 알림이 정상 수신되는지 확인
- **연락처 업데이트**: 학교 시스템의 다른 연락처 정보도 함께 업데이트
- **동료 안내**: 필요시 동료들에게 변경된 연락처 안내

## 문제 해결

### 정보 변경이 되지 않는 경우

1. **입력 형식 확인**: 이메일, 전화번호 등의 형식이 올바른지 확인
2. **네트워크 상태**: 인터넷 연결 상태 확인
3. **브라우저 새로고침**: 페이지를 새로고침하여 다시 시도
4. **관리자 문의**: 문제가 지속되면 학교 관리자에게 문의

### 비밀번호 변경 실패 시

1. **현재 비밀번호 확인**: 현재 비밀번호를 정확히 입력했는지 확인
2. **새 비밀번호 조건**: 새 비밀번호가 보안 조건을 만족하는지 확인
3. **브라우저 캐시**: 브라우저 캐시를 삭제하고 다시 시도
4. **비밀번호 초기화**: 필요시 관리자에게 비밀번호 초기화 요청

정기적인 정보 관리를 통해 안전하고 효율적인 디지털 수업 환경을 유지하세요!
