# 5. 로그아웃

수업이 완료되었다면 보안과 시스템 안정성을 위해 올바른 로그아웃 절차를 따라주세요. 적절한 로그아웃은 다음 수업의 원활한 진행을 위해 매우 중요합니다.

## 로그아웃이 중요한 이유

포커스팡에서 로그아웃은 단순한 종료 과정이 아닌 시스템 안정성과 보안을 위한 필수 절차입니다:

- **보안 유지**: 다른 사람이 계정에 접근하는 것을 방지
- **시스템 안정성**: 중복 로그인으로 인한 시스템 오류 방지
- **정확한 모니터링**: 학생 모니터링 기능의 정상 작동 보장
- **세션 관리**: 서버 리소스 효율적 사용

## 로그아웃 절차

### 1단계: 설정 페이지 접속

홈 화면에서 **좌측 상단의 프로필 사진 또는 이름**을 클릭하면 설정 페이지로 이동합니다.

### 2단계: 로그아웃 실행

설정 페이지에서 **[로그아웃]** 버튼을 클릭하면 로그아웃이 완료됩니다.

## 로그아웃 시 주의사항

:::danger 필수 로그아웃 상황

다음 상황에서는 **반드시 로그아웃**해주세요:

### 수업 종료 후

- **매 수업 종료 시**: 수업이 끝나면 즉시 로그아웃
- **교시 간 이동 시**: 다른 교실로 이동하기 전 로그아웃
- **하루 수업 종료 시**: 모든 수업이 끝나면 완전히 로그아웃

### 보안이 필요한 상황

- **컴퓨터 공유 시**: 다른 선생님과 컴퓨터를 공유할 때
- **자리 비움 시**: 잠시라도 자리를 비울 때
- **공용 컴퓨터 사용 시**: 교무실 공용 컴퓨터 사용 후

:::

## 로그아웃하지 않을 때 발생하는 문제

### 시스템 오류

- **중복 로그인**: 같은 계정으로 여러 곳에서 동시 접속
- **모니터링 오류**: 학생 모니터링 기능이 정상 작동하지 않음
- **데이터 충돌**: 수업 데이터가 정확히 저장되지 않을 수 있음

### 보안 위험

- **무단 접근**: 다른 사람이 계정에 접근할 가능성
- **개인정보 노출**: 학생 정보 등이 노출될 위험
- **수업 방해**: 의도하지 않은 수업 개입 가능성

## 로그아웃 후 확인사항

### 완전한 종료 확인

- **브라우저 탭 닫기**: 포커스팡 관련 모든 탭 닫기
- **브라우저 종료**: 가능하면 브라우저 완전 종료
- **로그인 화면 확인**: 다시 접속 시 로그인 화면이 나타나는지 확인

### 다음 수업 준비

- **수업 자료 정리**: 다음 수업에 필요한 자료 미리 준비
- **장비 점검**: 컴퓨터 및 네트워크 상태 확인
- **시간 여유**: 다음 수업 시작 전 충분한 시간 확보

## 로그아웃 관련 문제 해결

### 로그아웃이 되지 않는 경우

1. **브라우저 새로고침**: F5 키를 눌러 페이지 새로고침
2. **다른 탭 확인**: 다른 탭에서 포커스팡이 열려있는지 확인
3. **브라우저 재시작**: 브라우저를 완전히 종료 후 재시작
4. **캐시 삭제**: 브라우저 캐시 및 쿠키 삭제

### 중복 로그인 문제 발생 시

1. **모든 브라우저 종료**: 포커스팡 관련 모든 브라우저 종료
2. **잠시 대기**: 5-10분 정도 대기 후 재접속
3. **다른 기기 확인**: 다른 컴퓨터에서 로그인되어 있는지 확인
4. **관리자 문의**: 문제가 지속되면 학교 관리자에게 문의

## 효율적인 로그아웃 습관

### 일일 루틴

- **수업 전**: 이전 세션이 완전히 종료되었는지 확인
- **수업 중**: 중간 휴식 시간에도 보안 상태 점검
- **수업 후**: 즉시 로그아웃 후 다음 수업 준비

### 주간 점검

- **계정 상태**: 정상적으로 로그인/로그아웃되는지 확인
- **비밀번호**: 정기적인 비밀번호 변경
- **브라우저**: 브라우저 업데이트 및 캐시 정리

### 보안 의식

- **습관화**: 로그아웃을 자연스러운 습관으로 만들기
- **교육**: 다른 선생님들과 보안 의식 공유
- **점검**: 정기적인 보안 상태 점검

올바른 로그아웃 습관을 통해 안전하고 효율적인 디지털 수업 환경을 유지하세요!
