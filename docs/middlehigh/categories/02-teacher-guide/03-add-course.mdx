# 3. 수업 만들기

로그인을 완료했다면 이제 디지털 수업을 위한 시간표를 작성해보세요. 체계적인 수업 관리를 통해 효율적이고 효과적인 수업을 진행할 수 있습니다.

## 수업 만들기의 중요성

포커스팡에서 수업을 만드는 것은 단순한 시간표 작성을 넘어서 체계적인 디지털 수업 환경을 구축하는 과정입니다:

- **수업 체계화**: 과목별, 학년별, 반별 수업을 체계적으로 관리
- **학생 관리**: 각 수업에 참여하는 학생들을 효율적으로 관리
- **교육 도구 활용**: 포커스타임, 퀴즈팡 등 다양한 교육 도구 활용
- **학습 성과 추적**: 수업별 학생 참여도 및 학습 성과 모니터링

## 기본 수업 추가하기

### 1단계: 수업 추가 시작하기

수업 시간표에서 **[+수업 추가]** 버튼을 클릭합니다.

<Image src="/img/kr/middlehigh/teacher/02-01.jpg" size="lg" />

### 2단계: 수업 정보 입력하기

다음 정보를 정확히 입력해주세요:

- **과목명**: 담당하는 과목 (예: 수학, 영어, 국어, 과학 등)
- **학년**: 대상 학년 (1학년, 2학년, 3학년)
- **반**: 대상 반 (1반, 2반, 3반 등)
- **요일**: 수업이 진행되는 요일
- **교시**: 수업 시간 (1교시, 2교시 등)

### 3단계: 수업 생성 완료

모든 정보를 입력한 후 **[만들기]** 버튼을 클릭하면 수업이 추가됩니다.

<Image src="/img/kr/middlehigh/teacher/02-02.jpg" size="lg" />

## 효율적인 수업 관리

### 여러 시간 수업 추가하기

같은 과목을 다른 요일이나 시간에 추가하고 싶은 경우:

**[+수업 시간 추가하기]** 버튼을 클릭하여 한 번에 여러 개의 수업을 추가할 수 있습니다.

:::tip 시간 관리 팁

- 같은 과목의 여러 시간을 한 번에 추가하면 시간을 절약할 수 있습니다
- 잘못 입력한 수업은 언제든지 삭제할 수 있으니 걱정하지 마세요
  :::

### 다른 학년/반 수업 추가하기

여러 학년이나 반을 담당하는 경우:

**[+다른 학년 반 추가하기]** 버튼을 클릭하여 다른 학년이나 반의 수업을 추가할 수 있습니다.

## 수업 수정 및 삭제

### 수업 정보 수정하기

#### 1단계: 수정할 수업 선택

메인 시간표에서 수정할 수업의 우측 상단의 ✏️**연필 아이콘**을 클릭합니다.

<Image src="/img/kr/middlehigh/teacher/02-03.jpg" size="lg" />

#### 2단계: 정보 수정 또는 삭제

- **수정**: 변경할 내용을 입력한 후 **[저장]** 클릭
- **삭제**: 수업을 완전히 제거하려면 **[삭제]** 클릭

<Image src="/img/kr/middlehigh/teacher/02-04.jpg" size="lg" />

## 수업 관리 모범 사례

### 수업 생성 시 고려사항

- **정확한 정보 입력**: 학생들이 쉽게 식별할 수 있도록 명확한 과목명 사용
- **일관된 명명 규칙**: 과목명과 반 표기를 일관되게 유지
- **시간표 확인**: 실제 학교 시간표와 일치하는지 확인
- **중복 방지**: 같은 시간대에 중복 수업이 없는지 확인

### 효율적인 수업 운영

- **정기적 업데이트**: 시간표 변경 시 즉시 반영
- **학생 안내**: 수업 코드를 학생들에게 명확히 안내
- **백업 계획**: 수업 자료와 설정을 정기적으로 백업
- **성과 모니터링**: 수업별 학생 참여도 정기적 확인

### 문제 해결 가이드

- **수업이 생성되지 않는 경우**: 모든 필수 정보가 입력되었는지 확인
- **학생이 수업을 찾을 수 없는 경우**: 수업 코드와 정보가 정확한지 확인
- **시간표가 복잡한 경우**: 과목별로 색상을 다르게 설정하여 구분
- **수업 관리가 어려운 경우**: 비슷한 수업들을 그룹화하여 관리

## 다음 단계

수업 생성을 완료했다면:

1. **학생들에게 수업 코드 안내**
2. **첫 번째 수업 자료 준비**
3. **포커스타임 기능 탐색**
4. **퀴즈팡으로 학습 평가 준비**

체계적인 수업 관리를 통해 더욱 효과적인 디지털 수업을 진행해보세요!
