# 3. 수업 만들기

로그인을 완료했다면 이제 디지털 수업을 위한 시간표를 작성해 보세요.

## 수업 추가하기

### 1단계: 수업 추가 시작하기

수업 시간표에서 **[+수업 추가]** 버튼을 클릭합니다.

<Image src="/img/kr/middlehigh/teacher/02-01.jpg" size="lg" />

### 2단계: 수업 정보 입력하기

팝업 창에서 **과목명 / 학년 / 반 / 요일 / 교시**를 입력합니다.

- **과목명**: 국어, 수학, 영어 등
- **학년**: 몇 학년인지
- **반**: 몇 반인지
- **요일**: 언제 수업하는지
- **교시**: 몇 교시에 수업하는지

<Image src="/img/kr/middlehigh/teacher/02-02.jpg" size="lg" />

### 3단계: 수업 생성 완료

모든 정보를 입력한 후 **[만들기]** 버튼을 클릭하면 수업이 추가됩니다.

## 효율적인 수업 관리

### 여러 시간 수업 추가하기

같은 과목을 다른 요일이나 시간에 추가하고 싶은 경우 **[+수업 시간 추가하기]** 버튼을 클릭하여 한 번에 여러 개의 수업을 추가할 수 있습니다.

### 다른 학년/반 수업 추가하기

여러 학년이나 반을 담당하는 경우 **[+다른 학년 반 추가하기]** 버튼을 클릭하여 다른 학년이나 반의 수업을 추가할 수 있습니다.

:::tip

잘못 입력한 수업은 언제든지 삭제할 수 있으니 걱정하지 마세요!

:::

## 수업 수정 및 삭제

### 1단계: 수정할 수업 선택

메인 시간표에서 수정할 수업의 우측 상단의 ✏️**연필 아이콘**을 클릭합니다.

<Image src="/img/kr/middlehigh/teacher/02-03.jpg" size="lg" />

### 2단계: 정보 수정 또는 삭제

- **수정**: 변경할 내용을 입력한 후 **[저장]** 클릭
- **삭제**: 수업을 완전히 제거하려면 **[삭제]** 클릭

<Image src="/img/kr/middlehigh/teacher/02-04.jpg" size="lg" />

:::tip

- 수업 시간이 바뀌면 바로 수정해주세요
- 더 이상 하지 않는 수업은 삭제해주세요
- 수업 이름을 학생들이 이해하기 쉽게 지어주세요

:::

## 문제가 생겼을 때

- 페이지를 새로고침해보세요
- 다시 로그인해보세요
- 관리자 선생님에게 문의해보세요
