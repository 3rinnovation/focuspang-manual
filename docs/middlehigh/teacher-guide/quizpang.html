퀴즈팡

📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [퀴즈팡] 클릭​

퀴즈팡은 실시간 상호작용 기반의 학습 평가 도구로, 학생들의 이해도를 즉시 확인하고 학습 동기를 향상시킬 수 있는 효과적인 교육 도구입니다. 게임화된 학습 경험을 통해 학생들의 적극적인 참여를 유도할 수 있습니다.

언제 퀴즈팡을 사용하나요?
  • 형성평가: 수업 중 학습 내용 이해도 실시간 확인
  • 총괄평가: 단원 마무리 시 전체 학습 성과 평가
  • 복습 활동: 이전 수업 내용 복습 및 기억 강화
  • 워밍업: 수업 시작 전 주의집중 및 동기 유발

퀴즈팡 운영 방법​

1단계: 퀴즈 생성​

[+퀴즈 만들기] 버튼을 클릭하여 새로운 퀴즈를 생성합니다.

2단계: 퀴즈 유형 선택​

퀴즈 생성 팝업에서 [퀴즈]를 선택합니다.

3단계: 퀴즈 내용 작성​

다음 항목들을 체계적으로 작성합니다.

  • 퀴즈 이름: 학생들이 쉽게 식별할 수 있는 명확한 제목
  • 문제 유형: 객관식, 주관식, O/X 등 적절한 문제 형태 선택
  • 문제: 학습 목표에 부합하는 명확하고 구체적인 문제 작성
  • 지문: 필요시 문제 이해를 돕는 배경 정보 제공
  • 보기: 객관식 문제의 경우 적절한 선택지 작성
  • 제한 시간: 문제 난이도와 학생 수준을 고려한 적절한 시간 설정
  • 문제 해설: 정답 설명 및 학습 포인트 제공 (선택사항)
  • 카테고리: 체계적인 문제 관리를 위한 분류 (선택사항)

4단계: 퀴즈 실행​

퀴즈 생성 완료 후 자동으로 퀴즈팡 관리 화면으로 이동합니다. [퀴즈 시작] 버튼을 클릭하면 학생들이 퀴즈에 참여할 수 있습니다.

주의사항

포커스타임과 동시 실행 제한​

  • 포커스타임(수업)과 퀴즈팡은 동시에 실행할 수 없습니다.
  • 반드시 수업을 종료한 후 퀴즈팡을 시작해주세요.

퀴즈 종료 필수​

  • 퀴즈 완료 후 반드시 [종료] 버튼을 클릭해야 합니다.
  • 종료하지 않으면 학생들이 퀴즈 화면에서 나올 수 없습니다.
