수업 공지

수업 공지는 학생들에게 중요한 정보를 효과적으로 전달하는 핵심 소통 도구입니다. 체계적인 공지 관리를 통해 학생들의 학습 준비도를 높이고 원활한 수업 진행을 지원할 수 있습니다.

📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [수업 공지] 클릭​

수업 공지의 중요성​

효과적인 정보 전달​

수업과 관련된 중요한 정보를 모든 학생에게 동시에 정확하게 전달할 수 있습니다.

학습 준비도 향상​

사전 공지를 통해 학생들이 수업에 필요한 준비물이나 예습 내용을 미리 준비할 수 있도록 도와줍니다.

투명한 소통​

공지 내용과 읽음 확인 기능을 통해 교사와 학생 간의 투명하고 체계적인 소통이 가능합니다.

공지 작성 및 관리​

공지 등록​

1단계: 공지 작성 시작​

[+ 수업 공지] 버튼을 클릭하여 새로운 공지 작성 화면으로 이동합니다.

2단계: 공지 내용 작성​

다음 항목들을 체계적으로 작성합니다:

  • 제목: 공지 내용을 명확하게 표현하는 구체적인 제목
  • 공지 공개 일시: 공지 공개 시점 설정
  • 내용: 학생들에게 전달하고자 하는 상세한 정보
  • 첨부 자료: 필요시 관련 파일이나 이미지 추가

[등록] 버튼을 클릭하면 공지가 등록됩니다.

공지 공개 시점 설정

즉시 공개​

  • 공지를 등록하는 즉시 학생들에게 공개됩니다
  • 긴급한 안내사항이나 당일 수업 관련 정보에 적합합니다

예약 공개​

  • 원하는 날짜와 시간을 지정하여 공지를 예약할 수 있습니다
  • 미리 준비된 공지를 적절한 시점에 자동으로 공개하고 싶을 때 유용합니다
다중 반 공지 기능

[+ 공지할 반 추가하기] 버튼을 활용하면:

  • 현재 작성 중인 내용을 여러 반에 동시에 공지 가능
  • 같은 과목을 여러 반에서 가르치는 경우 효율적
  • 학년 공통 행사나 공지사항 전달 시 유용

공지 수정 및 삭제​

공지 내용 수정​

공지 목록에서 수정하려는 공지의 [내용 보기] 버튼을 클릭하여 상세 화면으로 이동한 후 내용을 수정할 수 있습니다.

공지 삭제​

공지 목록에서 삭제하려는 공지의 [삭제] 버튼을 클릭하여 공지를 삭제할 수 있습니다.

공지 효과 모니터링​

읽음 확인 기능​

공지 상세 화면에서 다음 정보를 확인할 수 있습니다:

  • 읽은 학생 목록: 공지를 확인한 학생들의 명단
  • 읽지 않은 학생 목록: 아직 공지를 확인하지 않은 학생들의 명단
  • 읽음 확인 시간: 각 학생이 공지를 확인한 시간
읽음 확인 활용법
  • 중요 공지 추적: 중요한 공지사항의 경우 미확인 학생에게 개별 안내
  • 참여도 파악: 학생들의 공지 확인 패턴을 통한 참여도 분석
  • 후속 조치: 미확인 학생에 대한 추가 안내 방법 결정

효과적인 공지 작성 가이드​

공지 제목 작성 원칙​

명확성​

  • 공지 내용을 한눈에 파악할 수 있는 구체적인 제목
  • 예: "3월 15일 수학 시험 범위 안내" (좋은 예)
  • 예: "시험 관련" (나쁜 예)

긴급성 표시​

  • 긴급한 공지의 경우 제목에 [긴급] 또는 [중요] 표시
  • 예: "[긴급] 내일 수업 시간 변경 안내"

날짜 포함​

  • 특정 날짜와 관련된 공지는 제목에 날짜 포함
  • 예: "4월 20일 현장학습 준비물 안내"

공지 내용 작성 요령​

구조화된 내용​

  • 언제: 날짜와 시간을 명확히 표시
  • 어디서: 장소가 관련된 경우 구체적인 위치 안내
  • 무엇을: 준비물이나 과제 내용을 상세히 설명
  • 어떻게: 실행 방법이나 절차를 단계별로 안내
  • 왜: 해당 공지의 목적이나 중요성 설명

읽기 쉬운 형식​

  • 긴 내용은 번호나 불릿 포인트로 구분
  • 중요한 내용은 굵은 글씨로 강조
  • 필요시 표나 목록 형태로 정리

첨부 자료 활용​

다양한 형태의 자료​

  • 이미지: 시각적 설명이 필요한 경우
  • 문서 파일: 상세한 안내서나 양식
  • 링크: 관련 웹사이트나 온라인 자료

파일 관리​

  • 파일명은 내용을 알 수 있도록 명확하게 작성
  • 파일 크기는 적절한 수준으로 조절
  • 학생들이 쉽게 열어볼 수 있는 형식 사용

공지 관리 모범 사례​

정기적인 공지 패턴​

주간 공지​

  • 매주 정해진 요일에 다음 주 수업 계획 안내
  • 과제 제출 마감일 리마인드
  • 시험 일정 및 준비 사항 안내

수업별 공지​

  • 수업 시작 전 당일 수업 내용 미리보기
  • 준비물이나 교재 안내
  • 특별 활동이나 실험 관련 안내

학생 참여 독려​

피드백 요청​

  • 공지 내용에 대한 학생들의 의견이나 질문 수렴
  • 이해도 확인을 위한 간단한 질문 포함

상호작용 촉진​

  • 학생들이 댓글이나 반응을 남길 수 있는 환경 조성
  • 공지 확인 후 간단한 응답 요청

공지 효과 분석​

정기적인 점검​

  • 공지 읽음률 분석을 통한 효과성 평가
  • 학생들의 피드백을 바탕으로 한 개선 방안 도출

개선 방안 적용​

  • 읽음률이 낮은 공지의 원인 분석
  • 더 효과적인 공지 방식 모색 및 적용
