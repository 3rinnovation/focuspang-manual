# 포커스팡 시작하기

안녕하세요!

포커스팡은 수업 시간에 자신의 주의집중 상태를 스스로 확인하고, 실시간으로 선생님과 함께 활동할 수 있는 **SEL(사회정서학습) 도구**입니다.
예를 들어, 주의집중 온도계, 실시간 주의집중 체크 같은 다양한 수업 활동을 제공합니다.

아래에서 여러분에게 맞는 안내(관리자, 선생님, 학생)를 골라 눌러 포커스팡을 바로 시작해 보세요!
