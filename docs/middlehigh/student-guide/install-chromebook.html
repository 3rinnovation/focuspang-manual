크롬북(Chromebook) 설치

  • 크롬 확장 프로그램을 설치합니다. 동영상 가이드(🔗바로가기)를 참조하세요.

  • 선생님이 CEU를 이용하여 크롬 확장 프로그램을 모든 학생에게 강제 설치하고 학생이 삭제하지 못하도록 고정합니다.
