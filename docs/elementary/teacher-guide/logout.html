6. 로그아웃

포커스팡 사용을 마치신 후에는 반드시 로그아웃을 해주세요. 로그아웃은 계정 보안과 서비스 안정성을 위해 매우 중요해요.

로그아웃 절차​

1단계: 설정 페이지 접속​

홈 화면에서 좌측 상단의 프로필 사진 또는 이름을 클릭하면 설정 페이지로 이동합니다.

2단계: 로그아웃 실행​

설정 페이지에서 [로그아웃] 버튼을 클릭하면 로그아웃이 완료됩니다.

로그아웃을 꼭 해주세요​

주의

로그아웃하지 않으면 중복 로그인이 발생할 수 있으며, 이 경우 학생 모니터링 기능이 정상적으로 작동하지 않을 수 있습니다.

다음 상황에서는 반드시 로그아웃해 주세요.

  1. 수업 종료 후
  • 매 수업 종료 시: 수업이 끝나면 즉시 로그아웃
  • 교시 간 이동 시: 다른 교실로 이동하기 전 로그아웃
  • 하루 수업 종료 시: 모든 수업이 끝나면 완전히 로그아웃
  1. 보안이 필요할 때
  • 컴퓨터 공유 시: 다른 선생님과 컴퓨터를 공유할 때
  • 자리 비움 시: 잠시라도 자리를 비울 때
  • 공용 컴퓨터 사용 시: 교무실 공용 컴퓨터 사용 후

문제 해결​

로그아웃이 되지 않는 경우​

  1. 브라우저 새로고침: F5 키를 눌러 페이지 새로고침
  2. 다른 탭 확인: 다른 탭에서 포커스팡이 열려있는지 확인
  3. 브라우저 재시작: 브라우저를 완전히 종료 후 재시작
  4. 캐시 삭제: 브라우저 캐시 및 쿠키 삭제

중복 로그인 문제 발생 시​

  1. 모든 브라우저 종료: 포커스팡 관련 모든 브라우저 종료
  2. 잠시 대기: 5-10분 정도 대기 후 재접속
  3. 다른 기기 확인: 다른 컴퓨터에서 로그인되어 있는지 확인
  4. 관리자 문의: 문제가 지속되면 학교 관리자에게 문의
