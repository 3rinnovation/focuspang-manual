# 2. 로그인

포커스팡 관리자로 로그인하는 방법을 알아봐요. 관리자는 학교에서 포커스팡을 사용하는 선생님들의 계정을 관리하는 중요한 역할을 해요.

## 관리자 계정이 뭐예요?

관리자 계정은 학교에서 포커스팡을 총괄 관리하는 특별한 계정이에요. 관리자는:

- 선생님들의 계정 가입 승인을 해줘요
- 선생님들의 계정 정보를 관리해요
- 필요할 때 다른 선생님에게도 관리자 권한을 줄 수 있어요

주로 정보부장 선생님이나 컴퓨터 담당 선생님이 관리자가 되세요.

## 관리자 로그인 방법

### 1단계: 관리자 페이지에 접속하기

포커스팡 관리자 어드민([🔗바로가기](http://admin.focuspang.com))에 접속해요.

### 2단계: 관리자 계정으로 로그인하기

발급받은 관리자 계정으로 로그인해요.

<Image src="/img/kr/elementary/admin/02-01.jpg" size="lg" />

:::tip 관리자 계정 발급 안내

1. 관리자 계정에 로그인하려면 **먼저 본사로부터 관리자 계정을 발급**받아야 해요.
2. 관리자 계정 신청 링크 **[🔗바로가기](https://focuspang.com/school.html)**
3. 관리자 계정은 **한 학교 당 1개의 계정만 발급**돼요.
   (관리자 계정을 발급받은 선생님이 전근 또는 퇴직으로 인해 학교에서 더이상 근무하지 않으신다면, 새로운 관리자 계정을 발급받을 수 있어요. 이 경우 **본사로 문의**주세요.)
   :::

## 로그인에 문제가 있을 때

로그인이 안 되면:

- 계정 정보가 정확한지 확인해보세요
- 인터넷 연결 상태를 확인해보세요
- 본사에 문의해서 도움을 받으세요
