# 4. 사용 가이드

포커스팡 관리자로서 더 자세한 사용법을 알고 싶으시다면, 동영상 가이드를 확인해보세요. 실제 화면을 보면서 따라하실 수 있어요.

## 동영상 가이드 확인하기

관리자용 포커스팡에 대한 자세한 사용법은 **동영상**을 통해 확인할 수 있어요.
동영상에서는 실제 관리자 화면을 보면서 단계별로 설명해드려요.

<Image src="/img/kr/elementary/admin/04-01.jpg" size="lg" />

## 자주 묻는 질문

**Q: 선생님 계정 승인은 얼마나 자주 확인해야 하나요?**
A: 새 학기나 새로운 선생님이 오실 때 자주 확인해주세요. 보통 주 1-2회 정도 확인하시면 좋아요.

**Q: 비밀번호를 잊어버린 선생님이 있을 때는 어떻게 하나요?**
A: 관리자 페이지에서 해당 선생님의 비밀번호를 초기화해주시면 돼요.

**Q: 관리자 권한을 다른 선생님에게 줄 수 있나요?**
A: 네, 관리자 등록 기능을 사용해서 다른 선생님에게도 관리자 권한을 줄 수 있어요.
