# 1. 관리자 역할

안녕하세요!
포커스팡 관리자는 학교에서 포커스팡을 사용하는 선생님들의 계정을 관리하는 역할을 해요.
주로 정보부장 선생님이나 컴퓨터 담당 선생님이 관리자가 되시는 경우가 많아요.

## 관리자가 할 수 있는 일

- **선생님 계정 관리**: 새로 가입한 선생님 계정을 승인하거나, 계정 정보를 수정할 수 있어요
- **관리자 추가**: 다른 선생님에게도 관리자 권한을 줄 수 있어요
- **계정 정지/삭제**: 필요한 경우 선생님 계정을 정지하거나 삭제할 수 있어요

:::tip 관리자와 선생님의 차이점
관리자는 **계정 관리**만 담당해요. 실제 수업(포커스타임, 퀴즈팡 등)은 각 선생님이 직접 하셔야 해요.
:::
