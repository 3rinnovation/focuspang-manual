# 1. 관리자 역할

- **선생님 계정**을 승인 및 수정, 정지, 삭제할 수 있습니다.
- 다른 교사를 관리자로 추가해 **관리자 권한**을 부여할 수 있습니다.

:::warning 참고

관리자는 **계정 및 학생 정보 관리에 집중**하며, 수업 관련 기능은 제공하지 않습니다.

:::
