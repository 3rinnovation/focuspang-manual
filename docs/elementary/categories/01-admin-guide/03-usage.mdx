# 3. 계정 관리

### 계정 추가

- **[관리자 등록]** 버튼을 클릭하여 선생님 또는 관리자 계정을 추가할 수 있습니다.
- 새로운 관리자는 최초 관리자와 동일한 권한을 가집니다.

<Image src="/img/kr/elementary/admin/03-01.jpg" size="lg" />

### 계정 관리

- 관리자 또는 선생님 계정을 **정지 / 삭제 / 정보 수정 / 비밀번호를 초기화**할 수 있습니다.
  - 계정 정지(예: 휴직)
  - 계정 삭제(예: 퇴직, 전근)

<Image src="/img/kr/elementary/admin/03-02.jpg" size="lg" />

- 학생 계정의 **정보(이름, 학번)를 수정**할 수 있습니다.
  <Image src="/img/kr/elementary/admin/03-03.jpg" size="lg" />

### 계정 승인

1. 선생님이 회원가입을 완료한 후, **관리자가 계정 승인을 해야** 선생님이 계정을 사용할 수 있습니다.

<Image src="/img/kr/elementary/admin/03-04.jpg" size="lg" />

2. **승인 대기 중인 계정**은 관리자 페이지의 **[상태]** 아이콘을 클릭하면 확인할 수 있으며, **계정 승인을 완료해야 사용 가능합니다.**

<Image src="/img/kr/elementary/admin/03-06.jpg" size="lg" />
