# 3. 계정 관리

관리자의 가장 중요한 역할은 학교에서 포커스팡을 사용하는 선생님들의 계정을 관리하는 것이에요. 계정 관리 방법을 알아봐요.

## 새로운 관리자 추가하기

### 왜 관리자를 추가하나요?

- 관리자가 휴직이나 전근을 가실 때를 대비해서
- 업무를 나누어서 더 효율적으로 관리하기 위해서
- 다른 선생님도 관리자 권한이 필요할 때

### 관리자 추가 방법

**[관리자 등록]** 버튼을 클릭하여 선생님 또는 관리자 계정을 추가할 수 있어요.
새로운 관리자는 최초 관리자와 동일한 권한을 가져요.

<Image src="/img/kr/elementary/admin/03-01.jpg" size="lg" />

## 계정 관리하기

### 계정 상태 변경하기

관리자 또는 선생님 계정을 **정지 / 삭제 / 정보 수정 / 비밀번호를 초기화**할 수 있어요.

- **계정 정지**: 선생님이 휴직하실 때
- **계정 삭제**: 선생님이 퇴직하거나 전근 가실 때
- **정보 수정**: 이름이나 기타 정보가 바뀌었을 때
- **비밀번호 초기화**: 선생님이 비밀번호를 잊어버리셨을 때

<Image src="/img/kr/elementary/admin/03-02.jpg" size="lg" />

### 학생 계정 관리하기

학생 계정의 **정보(이름, 학번)를 수정**할 수 있어요.

- 전학 온 학생의 정보 수정
- 잘못 입력된 학생 정보 수정

<Image src="/img/kr/elementary/admin/03-03.jpg" size="lg" />

## 선생님 계정 승인하기

### 왜 승인이 필요한가요?

- 학교 관련자만 포커스팡을 사용할 수 있도록 하기 위해서
- 보안을 위해서 관리자가 직접 확인하고 승인해요

### 승인 방법

1. 선생님이 회원가입을 완료한 후, **관리자가 계정 승인을 해야** 선생님이 계정을 사용할 수 있어요.

<Image src="/img/kr/elementary/admin/03-04.jpg" size="lg" />

2. **승인 대기 중인 계정**은 관리자 페이지의 **[상태]** 아이콘을 클릭하면 확인할 수 있으며, **계정 승인을 완료해야 사용 가능**해요.

<Image src="/img/kr/elementary/admin/03-06.jpg" size="lg" />
<Image src="/img/kr/elementary/admin/03-05.jpg" size="lg" />

:::tip 계정 관리 팁

- 정기적으로 승인 대기 중인 계정이 있는지 확인해보세요
- 퇴직하거나 전근 가신 선생님의 계정은 삭제해주세요
- 비밀번호를 잊어버린 선생님을 위해 초기화 기능을 활용해보세요
  :::

## 문제가 생겼을 때

계정 관리 중 문제가 생기면:

- 본사에 문의해서 도움을 받으세요
- 다른 관리자 선생님과 상의해보세요
- 사용법을 다시 확인해보세요
