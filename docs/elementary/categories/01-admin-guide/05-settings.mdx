# 5. 설정

관리자 계정의 설정을 변경하는 방법을 알아봐요. 계정 보안을 위해 정기적으로 비밀번호를 변경하는 것이 좋아요.

## 관리자 계정 설정하기

### 비밀번호 변경하기

왼쪽 상단의 톱니바퀴 아이콘을 클릭하면 계정 비밀번호를 변경할 수 있어요.

<Image src="/img/kr/elementary/admin/05-01.jpg" size="lg" />

### 언제 비밀번호를 변경해야 하나요?

- 처음 관리자 계정을 받았을 때
- 3-6개월마다 정기적으로
- 비밀번호가 유출되었다고 생각될 때
- 다른 사람이 내 계정을 사용했을 가능성이 있을 때

:::tip 안전한 비밀번호 만들기

- 8자 이상으로 만들어주세요
- 영문, 숫자, 특수문자를 섞어서 사용해주세요
- 생일이나 전화번호 같은 개인정보는 사용하지 마세요
- 다른 사이트에서 사용하는 비밀번호와 다르게 만들어주세요
  :::

## 계정 보안 관리하기

### 관리자 계정 보안 수칙

- 비밀번호를 다른 사람에게 알려주지 마세요
- 공용 컴퓨터에서 사용한 후에는 반드시 로그아웃하세요
- 이상한 활동이 발견되면 즉시 비밀번호를 변경하세요
- 정기적으로 계정 활동을 확인해보세요

### 문제가 생겼을 때

- 비밀번호를 잊어버렸을 때: 본사에 문의해주세요
- 계정이 해킹당했다고 생각될 때: 즉시 본사에 신고해주세요
- 설정 변경이 안 될 때: 브라우저를 새로고침하거나 다시 로그인해보세요
