# 5. 설정

관리자 계정 관리를 통해 안전하고 효율적으로 시스템을 운영할 수 있습니다.

## 계정 설정의 중요성

관리자 계정은 학교 전체의 포커스팡 시스템을 관리하는 중요한 계정입니다. 따라서 다음과 같은 이유로 정기적인 설정 관리가 필요합니다.

- **보안 강화**: 정기적인 비밀번호 변경으로 계정 보안 유지
- **개인정보 보호**: 최신 정보 유지로 개인정보 보호 강화
- **시스템 안정성**: 올바른 설정으로 시스템 오류 방지
- **업무 효율성**: 개인 맞춤 설정으로 업무 효율성 향상

## 비밀번호 변경

### 비밀번호 변경이 필요한 경우

- **최초 로그인**: 초기 비밀번호 사용 시
- **정기 변경**: 3개월마다 정기적으로 변경
- **보안 위험**: 계정 정보 유출 의심 시
- **공유 후**: 다른 사람과 계정을 공유한 후

### 비밀번호 변경 방법

왼쪽 상단의 **톱니바퀴 아이콘**을 클릭하면 계정 비밀번호를 변경할 수 있습니다.

<Image src="/img/kr/elementary/admin/05-01.jpg" size="lg" />

### 안전한 비밀번호 설정 가이드

#### 비밀번호 생성 원칙

- **길이**: 8자 이상 (12자 이상 권장)
- **복잡성**: 대문자, 소문자, 숫자, 특수문자 조합
- **고유성**: 다른 사이트와 다른 비밀번호 사용
- **예측 불가**: 개인정보와 관련 없는 내용

#### 피해야 할 비밀번호

- **개인정보 포함**: 이름, 생년월일, 전화번호 등
- **연속 문자**: 123456, abcdef 등
- **키보드 패턴**: qwerty, asdf 등
- **일반적 단어**: password, admin 등

#### 비밀번호 예시

- **좋은 예**: MySchool2024!@, Fp#Admin789
- **나쁜 예**: 123456, password, 홍길동1234

## 설정 관리 모범 사례

### 정기 점검

- **월 1회**: 계정 정보 확인 및 업데이트
- **분기 1회**: 비밀번호 변경
- **반기 1회**: 보안 설정 전체 점검
- **연 1회**: 시스템 환경 설정 재검토

### 보안 관리 팁

- **비밀번호 관리자 사용**: 안전한 비밀번호 생성 및 관리
- **이중 인증**: 가능한 경우 이중 인증 활성화
- **정기 로그 확인**: 비정상적인 접근 시도 모니터링
- **백업 연락처**: 여러 연락처 등록으로 소통 경로 확보

### 문제 예방

- **정보 최신화**: 개인정보 변경 시 즉시 업데이트
- **테스트 진행**: 설정 변경 후 정상 작동 확인
- **백업 계획**: 관리자 부재 시 대응 계획 수립
- **교육 이수**: 새로운 기능 출시 시 관련 교육 참여
