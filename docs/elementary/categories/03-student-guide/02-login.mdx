# 2. 로그인

포커스팡 앱을 설치했다면 이제 선생님 수업에 참여해보세요! 로그인하면 선생님과 함께 재미있는 수업 활동을 할 수 있어요.

## 로그인이 뭐예요?

로그인은 여러분이 선생님의 수업에 들어가는 과정이에요. 마치 교실 문을 열고 들어가는 것과 같아요. 로그인을 하면:

- 선생님이 내주는 과제를 확인할 수 있어요
- 퀴즈팡으로 재미있는 퀴즈를 풀 수 있어요
- 포커스 타임으로 집중해서 공부할 수 있어요
- 친구들과 함께 모둠 활동을 할 수 있어요

## 로그인하는 방법

### 1단계: 초대코드 로그인 선택하기

포커스팡을 실행한 후 나타나는 팝업 창에서 **[초대코드 로그인]** 버튼을 클릭해요.

<Image src="/img/kr/elementary/student/05-01.jpg" size="sm" />

### 2단계: 초대코드 입력하기

선생님에게 받은 **수업 초대코드를 입력**해요.

:::tip 초대코드 입력 팁

- 초대코드는 선생님이 알려주는 특별한 번호예요
- 영문으로 입력해야 해요
- 한글로 입력되면 한/영 변환키를 눌러서 영문으로 바꿔주세요
- 대문자와 소문자를 정확히 구분해서 입력해주세요
  :::

<Image src="/img/kr/elementary/student/05-02.jpg" size="lg" />

### 3단계: 학번과 이름 입력하기

초대코드와 함께 **학번 5자리**와 **이름**을 입력한 뒤 로그인해요.

:::tip 학번 입력하는 방법
학번은 **숫자 5개**만 입력해 주세요.
예를 들어, 1학년 1반 8번 학생이라면:

- 10108 또는 01108처럼 입력하면 돼요
- 학교에서 알려준 학번 형식에 맞게 입력해 주세요
- 이름은 한글로 정확히 입력해주세요
  :::

<Image src="/img/kr/elementary/student/05-03.jpg" size="lg" />

## 로그인이 잘 안 될 때

### 자주 생기는 문제들

- **초대코드가 틀렸을 때**: 선생님께 다시 확인해보세요
- **학번이 틀렸을 때**: 학번을 정확히 5자리로 입력했는지 확인해보세요
- **이름이 틀렸을 때**: 이름을 정확히 한글로 입력했는지 확인해보세요
- **인터넷이 안 될 때**: 와이파이가 연결되었는지 확인해보세요

### 해결 방법

1. 입력한 정보를 다시 한 번 확인해보세요
2. 앱을 끄고 다시 켜보세요
3. 그래도 안 되면 선생님께 도움을 요청하세요

## 로그인 성공!

로그인이 성공하면 여러분의 시간표가 나타나요. 이제 선생님과 함께 재미있는 수업을 시작할 수 있어요!
