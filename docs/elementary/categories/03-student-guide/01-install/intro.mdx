# 1. 설치 방법

안녕하세요!
포커스팡은 수업 시간에 선생님과 함께 재미있게 공부할 수 있는 앱이에요.
선생님이 내주시는 퀴즈도 풀고, 집중도 체크도 할 수 있어요.

포커스팡 학생용은 안드로이드 태블릿, 아이패드, 윈도우 컴퓨터, 크롬북에서 사용할 수 있어요.
아래에서 여러분이 사용하는 기기를 선택해 주세요.
