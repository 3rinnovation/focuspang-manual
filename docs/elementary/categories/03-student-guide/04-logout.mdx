# 4. 로그아웃

포커스팡 사용을 마쳤을 때는 로그아웃을 해서 내 계정을 안전하게 보호해야 해요. 로그아웃하는 방법을 배워봐요.

## 왜 로그아웃을 해야 하나요?

로그아웃을 하는 것은 중요해요. 왜냐하면:

- 다른 사람이 내 계정을 사용하지 못하게 해요
- 내 개인정보를 보호할 수 있어요
- 학교에서 공용 기기를 사용할 때 특히 중요해요
- 내 물건을 안전하게 보관하는 것과 같아요

학교에서 사물함을 잠그는 것처럼, 내 계정도 안전하게 보호해야 해요!

## 언제 로그아웃을 해야 하나요?

이럴 때 로그아웃을 해주세요:

- 포커스팡 사용을 모두 마쳤을 때
- 다른 학생이 같은 기기를 사용해야 할 때
- 학교 수업이 끝나고 집에 갈 때
- 컴퓨터를 떠나기 전에

## 로그아웃하는 방법

### 1단계: 설정 페이지로 가기

홈 화면에서 좌측 상단의 프로필 사진 또는 이름을 클릭하면 설정 페이지로 이동할 수 있어요.

### 2단계: 안전하게 로그아웃하기

설정 페이지에서 로그아웃 버튼을 눌러서 안전하게 로그아웃해요.

<Image src="/img/kr/elementary/student/12-01.jpg" size="lg" />

:::tip 로그아웃을 잊지 마세요

- 포커스팡 사용을 마쳤을 때는 항상 로그아웃해요
- 특히 학교 공용 기기를 사용할 때는 더욱 중요해요
- 로그아웃하면 내 계정이 안전하게 보호돼요
- 컴퓨터를 떠나기 전에 로그아웃하는 것을 잊지 마세요!
  :::

## 로그아웃하면 어떻게 되나요?

로그아웃을 하면:

- 로그인 화면으로 돌아가요
- 내 계정 정보가 안전하게 보호돼요
- 다음에 포커스팡을 사용할 때 다시 로그인해야 해요
- 내 수업 정보와 과제는 모두 안전하게 저장되어 있어요
