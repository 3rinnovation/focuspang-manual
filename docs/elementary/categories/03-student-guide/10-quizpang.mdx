# 10. 퀴즈팡

퀴즈팡은 선생님이 낸 퀴즈를 학생들이 동시에 푸는 퀴즈 배틀입니다.

1.  선생님이 퀴즈를 시작하면, 퀴즈가 바로 시작됩니다.

<Image src="/img/kr/elementary/student/10-01.jpg" size="lg" />

2. 학생들이 문제를 푼 후 **제한 시간**이 지나면 자동으로 순위가 매겨집니다.

<Image src="/img/kr/elementary/student/10-02.jpg" size="lg" />

3. **[닫기]** 버튼을 누르면 **퀴즈 결과 페이지로 이동**합니다.

<Image src="/img/kr/elementary/student/10-03.jpg" size="lg" />
