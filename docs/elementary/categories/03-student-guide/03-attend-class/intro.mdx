# 3. 수업 참여

선생님이 개설한 수업은 학생이 로그인하면 **자동**으로 시간표에 나타나므로, **별도로 수업을 추가할 필요가 없습니다.**

또한, 시간표에서 해당 수업을 클릭하면 공지, 과제 등을 확인할 수 있습니다.
