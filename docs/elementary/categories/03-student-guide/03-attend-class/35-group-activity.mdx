# 모둠 활동

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [모둠 활동] 클릭

## 모둠 활동이 뭐예요?

모둠 활동은 선생님이 반을 작은 그룹으로 나누어서 함께 활동하는 시간이에요.

모둠 활동에서는

- 몇 명의 친구들과 한 팀이 되어 활동해요.
- 서로 아이디어를 나누고 도와줄 수 있어요.
- 함께 문제를 풀거나 프로젝트를 만들어요.
- 모든 친구가 참여하고 기여할 수 있어요.

## 참여하는 방법

### 1단계: 내가 속한 모둠 확인하기

내가 속한 모둠을 확인하려면, **[모둠 상세 보기]** 버튼을 클릭합니다.

<Image src="/img/kr/elementary/student/11-01.jpg" size="lg" />

### 2단계: 모둠 친구들 만나기

내가 속한 그룹이 몇 조인지, 같은 조 학생은 누구인지 확인할 수 있습니다.
