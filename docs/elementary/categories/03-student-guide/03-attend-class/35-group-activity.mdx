# 모둠 활동

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [모둠 활동] 클릭

1. 내가 속한 모둠을 확인하려면, **[모둠 상세 보기]** 버튼을 클릭합니다.

   <Image src="/img/kr/elementary/student/11-01.jpg" size="lg" />

2. 내가 속한 그룹이 몇 조인지, 같은 조 학생은 누구인지 확인할 수 있습니다.

   <Image src="/img/kr/elementary/student/11-02.jpg" size="lg" />
