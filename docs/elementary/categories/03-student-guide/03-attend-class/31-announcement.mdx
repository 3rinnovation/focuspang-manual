# 수업 공지

선생님이 여러분에게 중요한 소식을 전해주는 기능이에요. 수업 공지를 확인하는 방법을 배워봐요.

## 수업 공지가 뭐예요?

수업 공지는 선생님이 반 전체에게 보내는 메시지예요. 선생님이 이런 것들을 알려주실 수 있어요:

- 숙제에 대한 안내
- 다음 시간에 준비할 것들
- 시험이나 중요한 일정
- 재미있는 활동 소개

## 수업 공지 확인하는 방법

### 1단계: 수업 공지 페이지로 가기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [수업 공지] 클릭**

### 2단계: 선생님 메시지 읽기

선생님이 올린 공지는 **[내용 보기]** 버튼을 눌러 확인할 수 있어요.

### 3단계: 내용 잘 이해했는지 확인하기

공지 내용을 읽고 잘 모르겠는 부분이 있으면 선생님이나 친구에게 물어보세요.

:::tip 왜 수업 공지를 확인해야 하나요?
선생님이 수업 공지를 통해 중요한 정보를 알려주세요. 자주 확인해서 놓치는 일이 없도록 해요! 집에서 편지를 확인하는 것처럼 말이에요.
:::
