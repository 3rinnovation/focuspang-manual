# 수업 공지

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [수업 공지] 클릭

선생님이 등록한 공지의 내용을 확인할 수 있습니다.

<Image src="/img/kr/elementary/student/07-01.jpg" size="lg" />

선생님이 올린 공지는 **[내용 보기]** 버튼을 눌러 확인할 수 있어요.
