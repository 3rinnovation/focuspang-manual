# 퀴즈팡

퀴즈팡은 선생님이 만든 문제를 푸는 재미있는 게임이에요. 게임을 하면서 공부할 수 있어요!

## 퀴즈팡이 뭐예요?

퀴즈팡은 배운 내용을 문제로 풀어보는 재미있는 방법이에요. 퀴즈팡을 할 때:

- 선생님이 수업 내용으로 문제를 만들어주세요
- 친구들과 누가 더 많이 맞히는지 경쟁할 수 있어요
- 내 점수를 확인하고 틀린 문제를 다시 공부할 수 있어요
- 친구들과 함께 하는 학습 게임 같은 거예요!

## 퀴즈팡 참여하는 방법

### 1단계: 퀴즈팡 페이지로 가기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [퀴즈팡] 클릭**

### 2단계: 선생님이 시작할 때까지 기다리기

선생님이 퀴즈를 시작하면, 퀴즈가 바로 시작돼요.

<Image src="/img/kr/elementary/student/10-01.jpg" size="lg" />

### 3단계: 문제 풀기

선생님이 정한 **제한 시간** 안에 문제를 빠르고 정확하게 풀어보세요.

### 4단계: 결과 확인하기

문제를 푼 후 **제한 시간**이 지나면 자동으로 친구들과 비교한 순위를 확인할 수 있어요.

<Image src="/img/kr/elementary/student/10-02.jpg" size="lg" />

### 5단계: 메인 화면으로 돌아가기

**[닫기]** 버튼을 누르면 **퀴즈 결과 페이지로 이동**해요.

<Image src="/img/kr/elementary/student/10-03.jpg" size="lg" />

:::tip 퀴즈팡 잘하는 방법

- 문제를 천천히 읽어보고 답해요
- 빨리 풀려고 하지만 실수하지 않도록 주의해요
- 수업 시간에 배운 내용을 떠올려봐요
- 친구들과 경쟁하며 재미있게 해요!
- 틀린 문제가 있어도 괜찮아요. 그렇게 배우는 거니까요!
  :::

## 퀴즈팡 끝난 후에는?

퀴즈팡을 마친 후에는:

- 틀린 문제에 대해 선생님에게 물어보세요
- 못 맞춘 내용을 다시 공부해보세요
- 다음 퀴즈를 위해 준비해요
- 잘한 것에 대해 스스로 칭찬해주세요!
