# 과제 관리

선생님이 내주신 과제를 확인하고 제출하는 방법을 배워봐요. 포커스팡으로 과제를 쉽게 제출할 수 있어요.

## 과제가 뭐예요?

과제는 선생님이 여러분에게 주시는 숙제예요. 과제를 통해서:

- 수업 시간에 배운 것을 연습할 수 있어요
- 선생님에게 내가 얼마나 잘 알고 있는지 보여줄 수 있어요
- 그 과목을 더 잘하게 될 수 있어요

선생님이 읽기, 쓰기, 문제 풀기 등 다양한 과제를 내주실 거예요.

## 과제 확인하고 제출하는 방법

### 1단계: 과제 페이지로 가기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [과제] 클릭**

### 2단계: 해야 할 과제 확인하기

1. 선생님이 내주신 과제를 확인해요
2. 해당 과제를 클릭해요

<Image src="/img/kr/elementary/student/08-01.jpg" size="lg" />

### 3단계: 과제 제출하기

**[제출하기]** 버튼을 누르면, 아래에 과제 제목 입력란과 **[과제 파일 업로드]** 버튼이 나타나요. 과제 제목을 작성하고 과제 파일을 업로드하면 **[제출완료] 버튼이 활성화**돼요.

### 4단계: 선생님에게 보내기

활성화된 **[제출 완료]** 버튼을 클릭하면 과제가 제출돼요.

<Image src="/img/kr/elementary/student/08-02.jpg" size="lg" />

:::tip 과제를 잘하는 방법

- 과제 설명을 두 번 읽어보고 시작해요
- 마감일 전에 미리 끝내요
- 파일이 잘 업로드되었는지 확인하고 제출해요
- 모르는 부분이 있으면 선생님에게 물어보세요
  :::

## 문제가 생겼을 때는?

파일을 업로드할 수 없거나 제출에 문제가 있으면 선생님에게 도움을 요청하세요. 선생님이 어떻게 하는지 알려주시거나 문제를 해결해 주실 거예요.
