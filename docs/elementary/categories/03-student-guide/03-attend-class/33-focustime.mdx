# 포커스 타임

포커스타임은 선생님과 함께하는 진짜 수업 시간이에요. 마치 교실에 있는 것처럼 선생님과 함께 공부할 수 있어요!

## 포커스타임이 뭐예요?

포커스타임은 선생님과 함께하는 진짜 수업 시간이에요. 포커스타임에서는:

- 선생님이 보여주시는 화면을 볼 수 있어요
- 화면에 메모를 할 수 있어요
- 선생님이 공유해주시는 자료를 받을 수 있어요
- 선생님이 허용해주신 앱과 웹사이트만 사용할 수 있어요 (공부에 집중할 수 있도록 도와줘요)

선생님이 바로 옆에서 도와주시는 것 같은 느낌이에요!

:::tip 포커스타임에서 무엇을 할 수 있나요?

- 선생님 화면을 크게 보면서 수업을 들을 수 있어요
- 화면에 펜으로 메모를 할 수 있어요
- 선생님이 보내주신 자료를 받아볼 수 있어요
- 선생님이 허용한 앱과 웹사이트만 사용할 수 있어요
  :::

## 포커스타임 참여하는 방법

### 1단계: 포커스타임 페이지로 가기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [포커스타임] 클릭**

### 2단계: 선생님 수업 듣기

선생님이 수업을 시작하면, 선생님의 화면이 여러분 화면에 자동으로 나타나요. 선생님이 보여주시는 모든 것을 볼 수 있어요!

### 3단계: 화면에 메모하기

🖌️**붓 아이콘**을 클릭하면 화면에 펜으로 메모를 할 수 있어요

- 🎨**색깔 선택**: 다양한 펜 색깔을 선택할 수 있어요
- ◽️**지우개**: 잘못 쓴 부분만 지울 수 있어요
- 🗑️**휴지통**: 쓴 내용을 모두 지울 수 있어요

<Image src="/img/kr/elementary/student/09-01.jpg" size="lg" />

### 4단계: 전체화면으로 보기

**[전체 화면 보기]** 버튼을 누르면, 선생님이 공유한 화면을 더 크고 선명하게 볼 수 있어요.

<Image src="/img/kr/elementary/student/09-02.jpg" size="lg" />
<Image src="/img/kr/elementary/student/09-03.jpg" size="lg" />

### 5단계: 사용할 수 있는 앱 확인하기

**[사용 가능한 앱과 웹사이트]**에서는 선생님이 수업 시간에 허용해주신 앱과 웹사이트 목록을 확인할 수 있어요.

<Image src="/img/kr/elementary/student/09-04.jpg" size="lg" />

### 6단계: 선생님이 주신 자료 받기

**[공유된 링크/파일 목록]**에서는 선생님이 공유한 링크들이 표시되며, 클릭하면 선생님이 주신 자료를 바로 열어볼 수 있어요.

<Image src="/img/kr/elementary/student/09-05.jpg" size="lg" />

## 포커스타임 잘 참여하는 방법

- 선생님이 보여주시는 것을 잘 봐요
- 중요한 내용이 나오면 메모해요
- 모르는 것이 있으면 질문해요
- 선생님이 허용해주신 앱만 사용해요
- 선생님과 친구들과 함께 즐겁게 공부해요!
