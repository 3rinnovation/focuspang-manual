# 2. 반 만들기

### 생성 - 담임 선생님

수업을 만들기 전에 먼저 학급의 **반**을 만들어야 합니다.\
반을 만들려면 먼저 학년과 반을 선택합니다. **반**이 숫자가 아니라면, **직접 입력** 체크박스를 선택하여 한글로 반 이름을 입력하세요.

<Image src="/img/kr/elementary/teacher/02-01.jpg" size="lg" />

만들어진 반이 제대로 등록되었는지 좌측 메뉴 상단에서 확인이 가능합니다.

<Image src="/img/kr/elementary/teacher/02-02.jpg" size="lg" />

### 생성 - 과목 선생님

**특수 교과목 선생님**의 경우, [반 만들기] 대신 **[교과목 선생님으로 시작하기]** 버튼을 클릭하여 수업을 만드세요.

:::danger 주의

특수 교과목 선생님이 반을 만들면, 모든 수업이 하나의 초대코드로 묶이게 되어 수업을 분리해서 운영할 수 없습니다.

따라서, 반드시 반을 만들지 않은 상태에서 수업을 **개별적으로 생성**해야 합니다.

:::

<Image src="/img/kr/elementary/teacher/02-03.jpg" size="lg" />

그 다음에 **[+수업 추가]** 버튼을 눌러 각각의 수업을 만들면 됩니다.

<Image src="/img/kr/elementary/teacher/02-04.jpg" size="lg" />

### 초대하기

반을 만들면 해당 반의 **초대코드**가 자동으로 생성됩니다. 이 초대코드는 학생들이 수업에 참여할 때 꼭 필요한 코드입니다.

- 학생들은 수업에 참여하려면 **초대코드, 학번, 이름**을 입력해야 해당 수업에 참여할 수 있습니다.
- 선생님이 시간표 내 어떤 수업을 추가하더라도, **같은 반의 초대코드는 동일**합니다.

<Image src="/img/kr/elementary/teacher/02-05.jpg" size="lg" />
