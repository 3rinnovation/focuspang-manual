# 2. 반 만들기

안녕하세요!
포커스팡을 사용하려면 먼저 선생님의 **반**을 만들어야 해요.
반을 만들어야 학생들이 초대코드로 수업에 참여할 수 있거든요.

선생님이 담임선생님인지, 과목선생님인지에 따라 방법이 조금 달라요.
아래에서 선생님에게 맞는 방법을 선택해 주세요.

## 담임 선생님이 반 만드는 방법

### 1단계: 학년과 반 선택하기

1. 먼저 **학년**을 선택해 주세요
2. 다음으로 **반**을 선택해 주세요
3. 만약 반 이름이 숫자가 아니라면 (예: 사랑반, 희망반), **직접 입력** 체크박스를 선택하고 한글로 반 이름을 입력해 주세요

<Image src="/img/kr/elementary/teacher/02-01.jpg" size="lg" />

### 2단계: 반이 잘 만들어졌는지 확인하기

반을 만들고 나면 화면 왼쪽 메뉴 상단에서 내가 만든 반이 잘 나타나는지 확인해 보세요.

<Image src="/img/kr/elementary/teacher/02-02.jpg" size="lg" />

## 과목 선생님이 수업 만드는 방법

과목 선생님(체육, 음악, 미술 등)은 반을 만들지 않고 바로 수업을 만들어야 해요.

:::tip 왜 과목 선생님은 반을 만들면 안 되나요?
과목 선생님이 반을 만들면 모든 수업이 하나의 코드로 묶여서, 1학년과 2학년 수업을 따로 관리할 수 없어요.
그래서 수업을 하나씩 따로 만들어야 해요.
:::

### 1단계: 교과목 선생님으로 시작하기

1. [반 만들기] 버튼 대신 **[교과목 선생님으로 시작하기]** 버튼을 눌러주세요

<Image src="/img/kr/elementary/teacher/02-03.jpg" size="lg" />

### 2단계: 수업 하나씩 추가하기

1. **[+수업 추가]** 버튼을 눌러주세요
2. 각 학년별, 반별로 수업을 하나씩 만들어 주세요

<Image src="/img/kr/elementary/teacher/02-04.jpg" size="lg" />

## 학생들을 수업에 초대하는 방법

반을 만들거나 수업을 만들고 나면, **초대코드**가 자동으로 만들어져요.
이 초대코드를 학생들에게 알려주면 학생들이 수업에 참여할 수 있어요.

### 초대코드가 무엇인가요?

- 초대코드는 학생들이 선생님 수업에 들어올 때 필요한 **비밀번호** 같은 거예요
- 학생들은 **초대코드 + 학번 + 이름**을 입력하면 수업에 참여할 수 있어요
- 담임선생님의 경우, 어떤 수업을 만들어도 **같은 반은 항상 같은 초대코드**를 사용해요
