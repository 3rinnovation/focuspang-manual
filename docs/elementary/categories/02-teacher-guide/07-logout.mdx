# 7. 로그아웃

포커스팡 사용을 마치신 후에는 반드시 로그아웃을 해주세요. 로그아웃은 계정 보안과 서비스 안정성을 위해 매우 중요해요.

## 왜 로그아웃을 해야 하나요?

로그아웃을 하는 것은 중요해요. 왜냐하면:

- 계정 보안을 지킬 수 있어요
- 중복 로그인을 방지할 수 있어요
- 학생 모니터링 기능이 정상적으로 작동해요
- 다른 선생님이 같은 기기를 사용할 수 있어요

## 로그아웃 하는 방법

### 1단계: 설정 페이지로 가기

홈 화면에서 좌측 상단의 프로필 사진 또는 이름을 클릭하면 설정 페이지로 이동해요.

### 2단계: 로그아웃하기

**[로그아웃]** 버튼을 누르면 로그아웃돼요.

<Image src="/img/kr/elementary/teacher/11-01.jpg" size="lg" />

:::tip 로그아웃을 꼭 해주세요

- 수업이 끝난 후에는 **반드시 로그아웃**해주세요
- 로그아웃하지 않으면 **중복 로그인**이 발생할 수 있으며, 이 경우 학생 모니터링 기능이 **정상적으로 작동하지 않을 수** 있어요
- 공용 컴퓨터를 사용하신다면 더욱 중요해요
  :::

## 언제 로그아웃을 해야 하나요?

이럴 때 로그아웃을 해주세요:

- 수업이 모두 끝났을 때
- 다른 기기에서 로그인해야 할 때
- 하루 업무가 끝났을 때
- 공용 컴퓨터를 사용하신 후

## 로그아웃 후에는?

로그아웃을 하면:

- 로그인 화면으로 돌아가요
- 계정 정보가 안전하게 보호돼요
- 다음에 사용할 때 다시 로그인하시면 돼요
- 모든 수업 정보와 설정은 그대로 저장되어 있어요
