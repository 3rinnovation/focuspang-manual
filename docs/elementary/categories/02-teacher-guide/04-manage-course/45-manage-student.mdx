# 학생 관리

학생의 학번과 이름을 수정하거나, 학생을 명단에서 삭제할 수 있습니다.

<Image src="/img/kr/elementary/teacher/08-01.jpg" size="lg" />

### 수정하기

✏️연필 아이콘을 클릭하면 학생의 학번과 이름을 수정할 수 있습니다.

<Image src="/img/kr/elementary/teacher/08-02.jpg" size="lg" />

### 내보내기

🚪문 모양 아이콘을 클릭하면 해당 학생을 명단에서 삭제할 수 있습니다.

<Image src="/img/kr/elementary/teacher/08-03.jpg" size="lg" />
