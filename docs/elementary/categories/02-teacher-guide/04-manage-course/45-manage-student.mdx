# 학생 관리

수업에 참여하는 학생들의 정보를 관리하는 방법을 알아봐요. 학생의 정보를 수정하거나 명단에서 제거할 수 있어요.

## 학생 관리가 왜 필요한가요?

학생 관리 기능을 사용하면:

- 학생 이름이나 학번을 잘못 입력했을 때 수정할 수 있어요
- 전학 간 학생을 명단에서 제거할 수 있어요
- 학생들의 정보를 정확하게 유지할 수 있어요
- 수업 참여 현황을 한눈에 볼 수 있어요

## 학생 관리 페이지 가는 방법

### 학생 명단 확인하기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [학생 명단] 클릭**

<Image src="/img/kr/elementary/teacher/08-01.jpg" size="lg" />

## 학생 정보 수정하기

### 언제 수정하나요?

- 학생이 이름을 잘못 입력했을 때
- 학번을 잘못 입력했을 때
- 학생 정보가 바뀌었을 때

### 수정하는 방법

✏️**연필 아이콘**을 클릭하면 학생의 학번과 이름을 수정할 수 있어요.

<Image src="/img/kr/elementary/teacher/08-02.jpg" size="lg" />

## 학생 명단에서 제거하기

### 언제 제거하나요?

- 학생이 전학을 갔을 때
- 학생이 더 이상 수업에 참여하지 않을 때
- 실수로 잘못 들어온 학생이 있을 때

### 제거하는 방법

🚪**문 모양 아이콘**을 클릭하면 해당 학생을 명단에서 삭제할 수 있어요.

<Image src="/img/kr/elementary/teacher/08-03.jpg" size="lg" />

:::tip 학생 관리 팁

- 학생 정보는 정확하게 유지해주세요
- 전학 간 학생은 바로 명단에서 제거해주세요
- 학생들이 이름을 잘못 입력하지 않도록 미리 안내해주세요
- 정기적으로 학생 명단을 확인해보세요
  :::

## 주의사항

- 학생을 명단에서 제거하면 되돌릴 수 없어요
- 학생을 제거하기 전에 정말 제거해야 하는지 다시 한 번 확인해주세요
- 학생 정보 수정은 언제든지 할 수 있어요
