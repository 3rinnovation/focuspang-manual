# 퀴즈팡

안녕하세요!
퀴즈팡은 선생님이 학생들에게 재미있는 퀴즈를 내주고, 학생들이 실시간으로 답을 할 수 있는 기능이에요.
수업이 끝난 후에 학생들의 이해도를 확인하거나, 복습용으로 사용하면 좋아요.

:::tip 언제 퀴즈팡을 사용하나요?

- 수업이 끝난 후 학생들이 얼마나 이해했는지 확인하고 싶을 때
- 지난 시간에 배운 내용을 복습하고 싶을 때
- 재미있게 문제를 풀면서 학습하고 싶을 때
  :::

## 퀴즈팡 사용하는 방법

### 퀴즈팡 메뉴로 이동하기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [퀴즈팡] 클릭**

### 1단계: 퀴즈 만들기 시작하기

1. **[+퀴즈 만들기]** 버튼을 클릭해 주세요

<Image src="/img/kr/elementary/teacher/06-01.jpg" size="lg" />

### 2단계: 퀴즈 종류 선택하기

2. 퀴즈 만들기 창이 나타나면 **[퀴즈]**를 선택해 주세요

<Image src="/img/kr/elementary/teacher/06-02.jpg" size="lg" />

### 3단계: 퀴즈 내용 작성하기

3. 다음 내용들을 차례대로 작성해 주세요:
   - **퀴즈 이름**: 학생들이 볼 퀴즈 제목
   - **문제 유형**: 객관식, 주관식 등 선택
   - **문제**: 학생들에게 낼 질문
   - **지문**: 문제를 이해하는 데 필요한 추가 설명 (선택사항)
   - **보기**: 객관식인 경우 선택지들
   - **제한 시간**: 학생들이 답할 수 있는 시간
   - **문제 해설**: 정답 설명 (선택사항)

<Image src="/img/kr/elementary/teacher/06-03.jpg" size="lg" />
<Image src="/img/kr/elementary/teacher/06-05.jpg" size="lg" />

### 4단계: 정답 설정하기

4. 각 보기 중에서 정답인 항목 오른쪽에 있는 **[정답]** 버튼을 눌러서 정답을 지정해 주세요

<Image src="/img/kr/elementary/teacher/06-04.jpg" size="lg" />

### 5단계: 학생들에게 퀴즈 내주기

5. 퀴즈를 다 만들면 퀴즈팡 관리 화면으로 자동으로 이동해요. **퀴즈 시작** 버튼을 클릭하면 학생들이 퀴즈를 풀 수 있어요

<Image src="/img/kr/elementary/teacher/06-06.jpg" size="lg" />

### 6단계: 학생들이 퀴즈 푸는 모습 확인하기

6. 퀴즈가 시작되면 학생들 화면에 문제가 나타나고, 학생들은 선생님이 정한 시간 안에 답을 선택하면 돼요

:::danger 주의

- 포커스타임(수업) 중에는 퀴즈팡을 동시에 할 수 없으니, **반드시 수업을 종료한 후 퀴즈팡을 시작해 주세요.**
- 퀴즈가 끝나면 선생님께서는 반드시 **[종료]** 버튼을 눌러야 학생들이 퀴즈 창에서 나올 수 있습니다.

:::
