# 4. 수업 진행

수업을 만들었다면 이제 본격적으로 수업을 진행해볼 차례예요! 포커스팡을 사용하면 학생들과 함께 다양한 활동을 할 수 있어요.

## 수업 진행 화면이 뭐예요?

수업 진행 화면은 선생님이 수업을 관리하고 학생들과 소통하는 중심 공간이에요. 여기서 할 수 있는 일들은:

- **수업 공지**: 학생들에게 중요한 알림을 전달해요
- **포커스 타임**: 학생들의 기기를 관리하고 집중도를 높여요
- **퀴즈팡**: 재미있는 퀴즈로 학습 효과를 확인해요
- **과제 관리**: 과제를 내주고 학생들의 제출을 확인해요
- **학생 관리**: 수업에 참여하는 학생들을 관리해요
- **모둠 활동**: 학생들을 그룹으로 나누어 협력 학습을 진행해요

## 수업 진행 화면으로 가는 방법

시간표에서 **진행하고 싶은 수업을 클릭**하면 수업 화면으로 이동해요.

## 수업 진행 시 도움이 되는 팁

### 수업 시작 전 준비사항

- 학생들에게 미리 포커스팡 앱을 설치하도록 안내해주세요
- 초대코드를 칠판에 적어두거나 학생들에게 미리 알려주세요
- 오늘 사용할 기능들을 미리 준비해두세요

### 효과적인 수업 진행 방법

- 수업 시작 시 공지사항을 먼저 확인해주세요
- 포커스 타임으로 학생들의 집중도를 관리해주세요
- 퀴즈팡으로 학생들의 이해도를 확인해주세요
- 모둠 활동으로 학생들의 참여도를 높여주세요

이제 각 기능별로 자세한 사용법을 알아보겠어요!
