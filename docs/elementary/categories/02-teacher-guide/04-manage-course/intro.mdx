# 4. 수업 진행

시간표에서 수업을 클릭하면, 수업 화면으로 이동합니다.

해당 화면에서는 수업 공지, 포커스 타임, 퀴즈팡, 과제 관리, 학생 관리, 모둠 활동에 대한 관리가 가능합니다.
