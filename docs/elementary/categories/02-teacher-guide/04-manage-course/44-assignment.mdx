# 과제 관리

#### 📍 메인 페이지 → 시간표에서 해당 수업 클릭 → 왼쪽 카테고리 [과제 관리] 클릭

### 등록

1. **[+ 과제 추가]** 버튼을 클릭하여 새로운 과제 등록 화면으로 이동합니다.

<Image src="/img/kr/elementary/teacher/07-01.jpg" size="lg" />

2. 제목, 과제 공개 일시, 내용을 작성한 후 과제를 등록합니다.

<Image src="/img/kr/elementary/teacher/07-02.jpg" size="lg" />

### 수정 및 삭제

등록된 과제는 [과제 관리] 리스트의 [더보기]를 클릭하여 내용을 **수정**하거나, [삭제] 버튼으로 과제를 **삭제**할 수 있습니다.

<Image src="/img/kr/elementary/teacher/07-03.jpg" size="lg" />

### 과제 확인

[과제 관리] 리스트의 [더보기]를 클릭하면 학생들의 과제 제출 상태를 **확인**하고, 제출된 과제를 **다운로드** 받을 수 있습니다.

<Image src="/img/kr/elementary/teacher/07-04.jpg" size="lg" />
