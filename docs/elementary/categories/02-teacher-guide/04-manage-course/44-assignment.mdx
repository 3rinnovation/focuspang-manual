# 과제 관리

안녕하세요!
과제 관리는 학생들에게 숙제나 과제를 내주고, 제출받을 수 있는 기능이에요.
학생들이 과제를 제출했는지 확인하고, 제출한 파일을 받아볼 수 있어요.

:::tip 과제 관리로 무엇을 할 수 있나요?

- 학생들에게 과제를 내줄 수 있어요
- 학생들이 과제를 제출했는지 확인할 수 있어요
- 제출된 과제 파일을 다운로드받을 수 있어요
- 과제 내용을 수정하거나 삭제할 수 있어요
  :::

## 과제 관리 사용하는 방법

### 과제 관리 메뉴로 이동하기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [과제 관리] 클릭**

### 1단계: 새 과제 만들기

1. **[+ 과제 추가]** 버튼을 클릭해서 과제 등록 화면으로 이동해 주세요

<Image src="/img/kr/elementary/teacher/07-01.jpg" size="lg" />

### 2단계: 과제 내용 작성하기

2. 다음 내용들을 입력해 주세요:
   - **과제 제목**: 학생들이 볼 과제 이름
   - **과제 내용**: 학생들이 해야 할 일을 자세히 써주세요
   - **공개 시점**: 언제 학생들에게 과제를 보여줄지 선택해 주세요

<Image src="/img/kr/elementary/teacher/07-02.jpg" size="lg" />

### 3단계: 과제 수정하거나 삭제하기

등록한 과제를 바꾸고 싶거나 지우고 싶을 때:

1. [과제 관리] 목록에서 **[더보기]** 버튼을 눌러주세요
2. **수정**하고 싶으면 내용을 바꿀 수 있어요
3. **삭제**하고 싶으면 [삭제] 버튼을 눌러주세요

<Image src="/img/kr/elementary/teacher/07-03.jpg" size="lg" />

### 4단계: 학생들 과제 제출 확인하기

학생들이 과제를 제출했는지 확인하고 싶을 때:

1. [과제 관리] 목록에서 **[더보기]** 버튼을 눌러주세요
2. 어떤 학생이 과제를 제출했는지 확인할 수 있어요
3. 제출된 과제 파일을 **다운로드**받을 수 있어요
