# 모둠 활동

1. **[+ 새 모둠 활동 주제 만들기]** 버튼을 클릭하면 모둠 주제 입력 팝업 창이 뜹니다.

<Image src="/img/kr/elementary/teacher/09-01.jpg" size="lg" />

2. 모둠 주제를 작성하고, 모둠별 인원수를 지정한 후 **[랜덤 뽑기]** 를 클릭하면 모둠이 자동으로 편성됩니다.

   :::tip

   - 일부 학생만 조를 변경하고 싶다면, 모둠 변경을 통해 변경할 수 있습니다.
   - 모둠을 직접 지정하고 싶다면, 랜덤 뽑기 후 각 학생별로 모둠 변경을 해주시면 됩니다.

   :::

<Image src="/img/kr/elementary/teacher/09-02.jpg" size="lg" />

3. 모둠이 만들어지면 **[모둠 상세 보기]** 버튼을 클릭해 각 조에 편성된 **학생 명단을 확인**할 수 있습니다.

<Image src="/img/kr/elementary/teacher/09-03.jpg" size="lg" />
<Image src="/img/kr/elementary/teacher/09-04.jpg" size="lg" />

4. 모둠별 학생은 [포커스 타임]에서 조별로 묶이고, 각 모둠별로 구분 지어 모니터링 할 수 있습니다.

<Image src="/img/kr/elementary/teacher/09-05.jpg" size="lg" />
