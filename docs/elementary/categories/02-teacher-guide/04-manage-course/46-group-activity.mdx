# 모둠 활동

#### 📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [모둠 활동] 클릭**

모둠 활동을 통해 학생들이 협력하며 학습할 수 있습니다.

## 모둠 만드는 방법

### 1단계: 새 모둠 활동 만들기

**[+ 새 모둠 활동 주제 만들기]** 버튼을 클릭하면 모둠 주제 입력 팝업 창이 나타납니다.

<Image src="/img/kr/elementary/teacher/09-01.jpg" size="lg" />

### 2단계: 모둠 설정하기

1. **모둠 주제**를 작성합니다. (예: "환경 보호 프로젝트", "과학 실험")
2. **모둠별 인원수**를 지정합니다. (보통 4-6명이 적당해요)
3. **[랜덤 뽑기]** 를 클릭하면 모둠이 자동으로 편성됩니다.

:::tip 모둠 편성 팁

- 일부 학생만 조를 변경하고 싶다면, **모둠 변경**을 통해 변경할 수 있어요
- 모둠을 **직접 지정**하고 싶다면, 랜덤 뽑기 후 각 학생별로 모둠 변경을 해주시면 돼요

:::

<Image src="/img/kr/elementary/teacher/09-02.jpg" size="lg" />

### 3단계: 모둠 확인하기

모둠이 만들어지면 **[모둠 상세 보기]** 버튼을 클릭해 각 조에 편성된 **학생 명단을 확인**할 수 있어요.

<Image src="/img/kr/elementary/teacher/09-03.jpg" size="lg" />
<Image src="/img/kr/elementary/teacher/09-04.jpg" size="lg" />

## 모둠 활동 진행하기

### 포커스 타임에서 모둠 모니터링

모둠별 학생은 [**포커스 타임**]에서 조별로 묶이고, 각 모둠별로 구분 지어 모니터링 할 수 있습니다.

<Image src="/img/kr/elementary/teacher/09-05.jpg" size="lg" />

### 모둠별 활동 관리

- 각 모둠의 진행 상황을 확인할 수 있어요.
- 모둠별로 다른 과제를 줄 수 있어요.
- 모둠 간 발표나 경쟁을 할 수 있어요.
