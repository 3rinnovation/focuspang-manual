# 11. 로그아웃

홈 화면에서 본인 이름을 클릭한 후, [로그아웃] 버튼을 누르면 로그아웃됩니다.

🚨 **주의**

- 수업이 끝난 후에는 **반드시 로그아웃**해주세요.
- 로그아웃하지 않으면 **중복 로그인**이 발생할 수 있으며, 이 경우 학생 모니터링 기능이 **정상적으로 작동하지 않을 수** 있습니다.
