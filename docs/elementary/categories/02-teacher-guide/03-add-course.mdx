# 3. 수업 만들기

### 수업 추가

1. 시간표 화면에서 **[+ 수업 추가]** 버튼을 클릭하세요.

<Image src="/img/kr/elementary/teacher/03-01.jpg" size="lg" />

2. 팝업 창에서 **과목명 / 학년 / 반 / 요일 / 교시**를 입력합니다.

3. **[수업 추가]** 버튼을 클릭하여 수업을 생성합니다.

   :::info

   다른 요일에도 같은 수업을 추가하고 싶다면, **[+ 수업 시간 추가하기]** 버튼을 클릭해 한 번에 여러 개의 수업을 추가할 수 있습니다.

   :::

<Image src="/img/kr/elementary/teacher/03-02.jpg" size="lg" />

### 수정 및 삭제

1. 메인 시간표에서 수정할 수업의 우측 상단의 **✏️연필 아이콘**을 클릭합니다.

<Image src="/img/kr/elementary/teacher/03-03.jpg" size="lg" />

2. 내용을 수정한 후 **[저장]** 버튼을 클릭하면 변경 사항이 저장되고, **[삭제]** 버튼을 클릭하면 해당 수업이 삭제됩니다.

<Image src="/img/kr/elementary/teacher/03-04.jpg" size="lg" />
