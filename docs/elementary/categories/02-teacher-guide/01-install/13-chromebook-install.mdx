# 크롬북에서 포커스팡 사용하기

:::info 참고해 주세요
크롬북은 별도 앱 설치 없이 웹 브라우저에서 바로 사용할 수 있어요.
:::

## 사용 방법

### 1단계: 크롬 브라우저 열기

크롬북에서 **크롬 브라우저**를 열어주세요. (크롬북에는 크롬 브라우저가 기본으로 설치되어 있습니다.)

### 2단계: 포커스팡 선생님 사이트 접속하기

주소창에 [teacher.focuspang.com](https://teacher.focuspang.com)을 입력하거나 링크를 클릭하여 사이트에 접속해 주세요.

### 3단계: 바로가기 만들기 (선택사항)

더 쉽게 사용하려면 바로가기를 만들어 보세요.

1. 포커스팡 사이트에 접속해 주세요.
2. 크롬 브라우저 오른쪽 위 **점 3개** 버튼을 클릭해 주세요.

<Image src="/img/kr/elementary/teacher/00-01.png" size="lg" />

3. **전송, 저장 및 공유** 를 클릭해 주세요.

<Image src="/img/kr/elementary/teacher/00-02.png" size="lg" />

4. **바로가기 만들기**를 클릭해 주세요.

<Image src="/img/kr/elementary/teacher/00-03.png" size="lg" />

5. **만들기**를 클릭해 주세요.

<Image src="/img/kr/elementary/teacher/00-04.png" size="lg" />

6. **시작 관리자 ⦿** 를 눌러 바로가기가 생성 되어 있는지 확인해 주세요.

<Image src="/img/kr/elementary/teacher/00-05.png" size="lg" />

## 주의사항

- 크롬 브라우저를 사용해 주세요.
- 인터넷 연결이 필요해요.
- 마이크와 카메라 권한을 허용해 주세요.

:::tip 도움말

- 사용이 어려우시면 학교 컴퓨터 담당 선생님께 도움을 요청해 주세요.
- 크롬북 CEU 설정에 대해 더 궁금하다면 해당 [**YouTube 영상**](https://www.youtube.com/watch?v=CLqLSEOcGyE&t=1247s)을 참고해 주세요.

:::
