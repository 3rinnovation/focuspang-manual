# 1. 설치

선생님이 사용하실 기기에 맞는 포커스팡을 설치해보세요.

포커스팡 선생님용은 **다양한 기기**에서 사용하실 수 있습니다.

왼쪽 사이드 바 또는 아래에서 선생님이 사용하실 기기를 선택해 주세요.

- [윈도우(MS Windows)](/docs/elementary/teacher-guide/install-windows)
- [안드로이드(Android)](/docs/elementary/teacher-guide/install-android)
- [크롬북 / iOS 기기](/docs/elementary/teacher-guide/install-chromebook)

:::info 참고사항

- 윈도우, 크롬북, iOS 기기에서는 앱 설치 없이 크롬 브라우저로 [**🔗홈페이지**](https://teacher.focuspang.com)에 접속하여 바로 사용할 수 있어요.
- 안드로이드 모바일 브라우저로는 원활한 이용이 어려우니 앱을 설치해 주세요.
- 아이패드용 선생님 앱은 곧 출시될 예정입니다.

  :::
