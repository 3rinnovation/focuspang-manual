# 윈도우 컴퓨터에 포커스팡 설치하기

:::info 참고해 주세요
따로 앱을 설치하지 않고, **크롬 브라우저**에서 [**🔗홈페이지**](https://teacher.focuspang.com)에 접속해서 바로 사용할 수도 있습니다.
:::

## 설치 방법

### 1단계: 파일 다운로드하기

<div class="fp-download-container fp-download-green">
  <div>
    <div class="fp-download-title">윈도우용 선생님 설치 파일</div>
    <div class="fp-download-desc">FocuspangTeacherSetup.exe</div>
  </div>
  <a
    class="fp-download-button"
    href="https://ss1.cloud.kt.com:1000/focuspang-media/downloads/FocuspangTeacherSetup.exe"
  >
    <img
      class="fp-download-icon"
      src="/img/download-white-icon.svg"
      alt="다운로드"
    />
    다운로드
  </a>
</div>

위 버튼을 눌러서 **설치 파일을 다운로드** 하거나, 플레이스토어에서 **포커스팡 선생님을 검색**해 주세요.

### 2단계: 파일 실행하기

1. 다운로드가 완료되면 **다운로드 폴더**를 열어주세요.
2. **FocuspangTeacherSetup.exe** 파일을 찾아주세요.
3. 파일을 더블클릭해서 실행해 주세요.

### 3단계: 설치하기

1. 설치 프로그램이 시작되면 다음 버튼을 눌러주세요.
2. 설치가 완료될 때까지 기다려주세요.
3. 완료 버튼을 누르면 설치가 완료돼요.

## 설치 완료 후

설치가 끝나면 바탕화면에 포커스팡 선생님용 아이콘이 생겨요.\
아이콘을 더블클릭하면 포커스팡을 사용할 수 있어요.

:::tip 도움말
설치가 어려우시면 학교 컴퓨터 담당 선생님께 도움을 요청해 주세요.
:::
