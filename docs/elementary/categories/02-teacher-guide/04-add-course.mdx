# 3. 수업 만들기

반을 만들었다면 이제 시간표에 수업을 추가해보세요. 수업을 만들면 학생들과 함께 포커스타임, 퀴즈팡 등의 활동을 할 수 있어요.

## 수업이 뭐예요?

수업은 선생님이 학생들과 함께 하는 학습 활동이에요. 포커스팡에서 수업을 만들면:

- 학생들의 기기를 관리할 수 있어요
- 퀴즈를 만들어서 함께 풀 수 있어요
- 과제를 내주고 확인할 수 있어요
- 수업 자료를 공유할 수 있어요

## 수업 추가하는 방법

### 1단계: 수업 추가 버튼 누르기

시간표 화면에서 **[+ 수업 추가]** 버튼을 클릭하세요.

<Image src="/img/kr/elementary/teacher/03-01.jpg" size="lg" />

### 2단계: 수업 정보 입력하기

팝업 창에서 **과목명 / 학년 / 반 / 요일 / 교시**를 입력해요.

- **과목명**: 국어, 수학, 영어 등
- **학년**: 몇 학년인지
- **반**: 몇 반인지
- **요일**: 언제 수업하는지
- **교시**: 몇 교시에 수업하는지

### 3단계: 수업 만들기

**[수업 추가]** 버튼을 클릭하여 수업을 생성해요.

:::tip 여러 요일에 같은 수업이 있나요?
다른 요일에도 같은 수업을 추가하고 싶다면, **[+ 수업 시간 추가하기]** 버튼을 클릭해 한 번에 여러 개의 수업을 추가할 수 있어요.
:::

<Image src="/img/kr/elementary/teacher/03-02.jpg" size="lg" />

## 수업 수정하고 삭제하기

### 수업 정보 수정하기

1. 메인 시간표에서 수정할 수업의 우측 상단의 **✏️연필 아이콘**을 클릭해요.

<Image src="/img/kr/elementary/teacher/03-03.jpg" size="lg" />

2. 내용을 수정한 후 **[저장]** 버튼을 클릭하면 변경 사항이 저장되고, **[삭제]** 버튼을 클릭하면 해당 수업이 삭제돼요.

<Image src="/img/kr/elementary/teacher/03-04.jpg" size="lg" />

:::tip 수업 관리 팁

- 수업 시간이 바뀌면 바로 수정해주세요
- 더 이상 하지 않는 수업은 삭제해주세요
- 수업 이름을 학생들이 이해하기 쉽게 지어주세요
  :::

## 수업을 만든 후에는?

수업을 만든 후에는:

- 학생들에게 초대코드를 알려주세요
- 포커스타임을 시작해서 학생들과 함께 수업해보세요
- 퀴즈팡으로 재미있는 퀴즈를 만들어보세요
- 과제를 내주고 학생들의 제출을 확인해보세요

## 문제가 생겼을 때

수업 만들기에 문제가 생기면:

- 페이지를 새로고침해보세요
- 다시 로그인해보세요
- 관리자 선생님에게 문의해보세요
