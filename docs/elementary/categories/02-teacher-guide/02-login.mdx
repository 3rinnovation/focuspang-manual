# 1. 회원가입 / 로그인

안녕하세요!
포커스팡을 사용하려면 먼저 선생님 계정을 만들어야 해요.
계정을 만들고 나면 학생들과 함께 재미있는 수업을 할 수 있어요.

## 선생님 계정 만들기 (회원가입)

### 1단계: 포커스팡 선생님 사이트 접속하기

🔗 [선생님 페이지 바로가기](https://teacher.focuspang.com/ko/login)

<Image src="/img/kr/elementary/teacher/01-02.jpg" size="lg" />

### 2단계: 회원가입 정보 입력하기

다음 순서대로 정보를 입력해 주세요:

1. **이메일 주소 입력** → 인증 코드가 이메일로 발송돼요
2. **이메일에서 인증 코드 확인** → 받은 코드를 입력해 주세요
3. **이름과 비밀번호 입력** → 기억하기 쉬운 비밀번호로 설정해 주세요
4. **약관 동의** → 포커스팡 이용 약관에 동의해 주세요
5. **소속 학교 선택** → 선생님이 근무하시는 학교를 선택해 주세요
6. **[확인하기], [다음] 버튼 클릭** → 회원가입이 완료돼요

:::tip 개인정보동의서는 필요 없어요
회원가입할 때 개인정보 동의를 받기 때문에, 별도로 개인정보동의서를 제출하실 필요가 없어요.
:::

### 3단계: 관리자 승인 받기

회원가입 후 바로 사용할 수는 없어요. 학교의 **관리자 선생님**이 계정을 승인해야 사용할 수 있어요.

:::tip 빠른 승인을 위한 팁
학교의 컴퓨터 담당 선생님이나 정보부장 선생님께 "포커스팡 계정 승인 부탁드립니다"라고 말씀해 주세요.
:::

### 로그인

회원가입 후 승인된 계정으로 로그인합니다.
