# 1. 회원가입 / 로그인

포커스팡을 사용하여 효과적인 디지털 수업을 진행하기 위해서는 먼저 선생님 계정으로 로그인해야 합니다.

## 계정 신청 및 회원가입

### 계정 신청 방법

<Image src="/img/kr/elementary/teacher/01-01.jpg" size="lg" />

### 회원가입 절차

회원가입은 다음과 같은 단계로 진행됩니다.

1. **이메일 인증**: 사용할 이메일 주소 입력 및 인증 메일 확인
2. **인증 코드 입력**: 이메일로 받은 인증 코드 입력
3. **개인정보 입력**: 이름, 비밀번호 등 기본 정보 입력
4. **약관 동의**: 서비스 이용약관 및 개인정보 처리방침 동의
5. **소속 학교 선택**: 근무하는 학교 선택
6. **가입 신청 완료**: [확인하기], [다음] 버튼 클릭 후 가입 신청 완료

:::tip 개인정보 동의서 관련 안내
회원가입 시 개인정보 수집 및 제3자 제공 동의를 받고 있으므로, 별도의 개인정보동의서를 제출할 필요가 **없습니다.**
:::

<Image src="/img/kr/elementary/teacher/01-02.jpg" size="lg" />

### 계정 승인 절차

1. **관리자 확인**: 회원 가입 신청 완료 후 학교 관리자가 신청 내역 확인
2. **승인 처리**: 관리자가 계정 승인 완료
3. **사용 가능**: 승인 완료 후 계정 사용 가능

:::tip 빠른 승인을 위한 팁
빠른 승인 처리를 위해 **학교 관리자 선생님께 직접 승인을 요청**해주세요.
:::

## 로그인

### 로그인 절차

1. **사이트 접속**: 선생님용 포커스팡 사이트 접속
2. **계정 정보 입력**: 가입 시 등록한 이메일과 비밀번호 입력
3. **로그인 완료**: 계정 확인 후 메인 화면으로 이동

<Image src="/img/kr/elementary/teacher/01-03.jpg" size="lg" />

### 자주 발생하는 문제

- **승인 대기**: 관리자 승인이 완료되지 않은 경우
- **비밀번호 오류**: 비밀번호를 잘못 입력한 경우
- **계정 정보 오류**: 이메일 주소를 잘못 입력한 경우
- **브라우저 문제**: 브라우저 호환성 문제

### 해결 방법

1. **승인 상태 확인**: 관리자에게 승인 상태 문의
2. **비밀번호 재설정**: 비밀번호 찾기 기능 이용
3. **계정 정보 확인**: 가입 시 사용한 정확한 이메일 주소 확인
4. **브라우저 설정**: Chrome 브라우저 사용 및 최신 버전 업데이트

## 최초 로그인 후 설정

### 프로필 설정

- **개인정보 확인**: 이름, 연락처 등 개인정보 정확성 확인
- **프로필 사진**: 학생들이 쉽게 식별할 수 있는 프로필 사진 설정
- **담당 과목**: 담당하는 과목 정보 입력

### 보안 설정

- **비밀번호 변경**: 보안을 위해 초기 비밀번호 변경 권장

### 수업 준비

- **시간표 작성**: 담당 수업 시간표 입력
- **수업 생성**: 첫 번째 수업 생성 및 설정
- **기능 탐색**: 주요 기능들 미리 살펴보기
