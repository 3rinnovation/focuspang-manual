# 수업 공지

안녕하세요!
수업 공지는 학생들에게 중요한 내용을 전달할 때 사용하는 기능이에요.
숙제 안내, 준비물 공지, 시험 일정 등을 학생들에게 알려줄 수 있어요.

:::tip 수업 공지로 무엇을 할 수 있나요?

- 학생들에게 중요한 내용을 전달할 수 있어요
- 사진이나 파일도 함께 보낼 수 있어요
- 여러 반에 동시에 같은 공지를 보낼 수 있어요
- 원하는 시간에 공지가 나타나도록 예약할 수 있어요
  :::

## 수업 공지 사용하는 방법

### 수업 공지 메뉴로 이동하기

📍 **메인 페이지** → **시간표에서 해당 수업 클릭** → **왼쪽 카테고리 [수업 공지] 클릭**

### 1단계: 새 공지 만들기 시작하기

1. **[+ 수업 공지]** 버튼을 클릭해서 공지 작성 화면으로 이동해 주세요

<Image src="/img/kr/elementary/teacher/04-01.jpg" size="lg" />

### 2단계: 공지 내용 작성하기

2. 다음 내용들을 입력해 주세요:
   - **제목**: 학생들이 볼 공지 제목
   - **내용**: 학생들에게 전달하고 싶은 내용
   - **공개 시점**: 언제 학생들에게 보여줄지 선택

:::info 공지 공개 시점 선택하기

- **즉시 공개**: 공지를 등록하자마자 학생들이 바로 볼 수 있어요
- **예약 공개**: 원하는 날짜와 시간을 정해서 그때 공지가 나타나도록 할 수 있어요

:::

### 3단계: 추가 기능 사용하기 (선택사항)

- **여러 반에 동시 공지**: [+ 공지할 반 추가하기] 버튼을 누르면 여러 반에 같은 내용을 한 번에 보낼 수 있어요
- **파일 첨부**: 사진이나 파일을 함께 보내고 싶으면 첨부할 수 있어요
- **[등록]** 버튼을 누르면 공지가 완성돼요
