# 3. 반 / 수업 만들기

포커스팡을 사용하려면 먼저 선생님의 **반과 수업**을 만들어야 합니다.
반을 만들어야 **학생들이 초대코드로 수업에 참여**할 수 있고, 이후 수업을 만들면 학생들과 함께 **포커스타임, 퀴즈팡 등의 활동**을 할 수 있습니다.

선생님이 담임 선생님인지, 과목 선생님인지에 따라 방법이 조금 다르므로 알맞은 선생님 역할을 선택해 주세요.

- [담임 선생님](/docs/elementary/teacher-guide/add-course-class-homeroom)
- [과목 선생님](/docs/elementary/teacher-guide/add-course-class-subject)
