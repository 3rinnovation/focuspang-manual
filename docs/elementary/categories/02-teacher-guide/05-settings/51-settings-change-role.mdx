# 선생님 역할(담임/과목) 선택

포커스팡에서는 선생님의 역할에 따라 사용할 수 있는 기능이 달라져요. 내 역할을 정확히 설정해서 포커스팡을 더 효과적으로 사용해보세요.

## 담임 선생님과 과목 선생님의 차이점

### 담임 선생님

- 반 전체를 관리하는 선생님이에요
- 학생들의 모든 과목을 관리할 수 있어요
- 반 단위로 수업을 만들고 관리해요

### 과목 선생님

- 특정 과목만 가르치는 선생님이에요
- 여러 반에서 같은 과목을 가르치실 수 있어요
- 과목별로 수업을 만들고 관리해요

## 선생님 역할 변경하는 방법

### 1단계: 설정 페이지로 가기

홈 화면에서 좌측 상단의 프로필 사진 또는 이름을 클릭하면 설정 페이지로 이동해요.

### 2단계: 역할 선택하기

담임 선생님 또는 과목 선생님으로 변경이 가능해요.

<Image src="/img/kr/elementary/teacher/10-02.jpg" size="lg" />

:::tip 어떤 역할을 선택해야 할까요?

- **담임 선생님**: 한 반의 모든 과목을 가르치시는 경우
- **과목 선생님**: 여러 반에서 특정 과목만 가르치시는 경우 (예: 영어, 음악, 미술 등)
  :::

## 역할 변경 시 주의사항

- 역할을 변경하면 수업 관리 방식이 달라져요
- 기존에 만든 수업들은 그대로 유지돼요
- 역할에 맞는 기능들을 사용하실 수 있어요

## 문제가 생겼을 때

역할 변경이 안 되거나 문제가 생기면:

- 페이지를 새로고침해보세요
- 다시 로그인해보세요
- 관리자 선생님에게 문의해보세요
