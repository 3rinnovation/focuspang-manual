Index for Student

📄️ 1. Android Installation

- You can check the installation process through the student Android installation guide video.\

📄️ 2. iPad Installation

- Download and install the student app (focuspang for Students) from the App Store.

📄️ 3. MS Windows Installation

- Download and install the Windows exe file.

📄️ 4. Chromebook Installation

- Install the Chrome extension. https://focuspang.com/dn.html#Google

📄️ 5. Login

- After launching FocusPang, click the \[Invitation Code Login] button.

📄️ 6. Adding Classes

- Click [+Add Class] next to the schedule on the main page.

📄️ 7. Class Announcements

- You can check the content of announcements registered by the teacher.

📄️ 8. Assignments

- You can check assignments registered by the teacher.

📄️ 9. Class Screen

- This is the window where you can see the screen shared by the teacher.

📄️ 10. Quizpang

- Quizpang is a quiz battle where students solve problems simultaneously when the teacher creates a quiz.

📄️ 11. Group Activity

- Click \[Group Activity] to go to the group activity page.

📄️ 12. Logout

- Click your name in the upper left corner to go to the settings page and log out.
